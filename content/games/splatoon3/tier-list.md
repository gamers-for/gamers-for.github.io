## 最強武器ランキング早見表

スプラトゥーン3の最強武器ランキング。各武器の性能・サブスペ構成・環境適正を考慮した総合評価。

### 評価基準

| 基準 | 内容 |
|------|------|
| **対象ルール** | ナワバリバトル・ガチエリア・ガチヤグラ・ガチホコ・ガチアサリの全ルール平均 |
| **重視項目** | 塗り能力とキル能力を同等に評価 |
| **マッチ想定** | バンカラマッチ全ルール・Xマッチを想定 |
| **編成考慮** | マッチングを考慮し編成に左右されにくい武器を高評価 |

### ティア早見表

| ティア | 概要 |
|--------|------|
| **S+** | 塗りもキルもできる武器。どのルールでもトップクラスで活躍 |
| **S** | S+に劣るが塗りもキルもどちらもできる |
| **A+** | 扱いやすく塗りまたはキルができる武器 |
| **A** | 塗りまたはキルに向く武器 |
| **B+〜C** | 扱いが難しく塗りに向かない武器。ただし使い手次第 |

**[ジムワイパー](../weapons/ジムワイパー/)** — 射程が長くキル性能が高い武器。タテ振り→ヨコ振りや、タテ振り→クイックボムのコンボは広い範囲でキルを狙える。

**[ジムワイパーヒュー](../weapons/ジムワイパーヒュー/)** — ワイパー。触れた敵の移動速度を落とし、インク回復を妨害する霧とカニ型タンクに搭乗して連射と範囲攻撃の構成。斬撃の判定が広いというワイパーの強みが光る。

**[H3リールガン](../weapons/h3リールガン/)** — シューター。当てた敵をマーキングして位置を把握と味方にスピードアップとインク回復のバフを付与の構成。扱いやすいというシューターの強みが光る。

**[ジェットスイーパー](../weapons/ジェットスイーパー/)** — シューター。直線状にマーキングする新サブウェポンと敵のインクを吸い込み、溜めて撃ち返すの構成。扱いやすいというシューターの強みが光る。

**[スペースシューター](../weapons/スペースシューター/)** — シューター。当てた敵をマーキングして位置を把握とメガホンから直線レーザーを発射の構成。扱いやすいというシューターの強みが光る。

**[ソイチューバー](../weapons/ソイチューバー/)** — チャージャー。壁に当たると分裂し広範囲にダメージを与える誘導弾と複数の敵をロックオンしてミサイルを発射の構成。超長射程というチャージャーの強みが光る。

**[ソイチューバーカスタム](../weapons/ソイチューバーカスタム/)** — チャージャー。転がすと3回バウンドして爆発するボムと巨大ハンコで前方を叩きつけながら進むの構成。超長射程というチャージャーの強みが光る。

**[スパイガジェット](../weapons/スパイガジェット/)** — シェルター。設置型の罠とサメに乗って突進し、着地点で爆発の構成。傘で弾を防げるというシェルターの強みが光る。

**[スパイガジェットソレーラ](../weapons/スパイガジェットソレーラ/)** — シェルター。壁に当たると分裂し広範囲にダメージを与える誘導弾と大きなシートを展開し、敵の視界を遮るの構成。傘で弾を防げるというシェルターの強みが光る。

**[クラッシュブラスター](../weapons/クラッシュブラスター/)** — ブラスター。投げて一定時間後に爆発する基本ボムと3発の強力な弾を発射する長射程スペシャルの構成。爆風で直撃しなくてもダメージというブラスターの強みが光る。

**[トライストリンガー](../weapons/トライストリンガー/)** — ストリンガー。触れた敵の移動速度を落とし、インク回復を妨害する霧とメガホンから直線レーザーを発射の構成。独特な弾道で奇襲可能というストリンガーの強みが光る。

**[トライストリンガーコラボ](../weapons/トライストリンガーコラボ/)** — ストリンガー。設置すると自動でインクを周囲に撒き続けると偽装イカを大量に撒いて敵を混乱させるの構成。独特な弾道で奇襲可能というストリンガーの強みが光る。
