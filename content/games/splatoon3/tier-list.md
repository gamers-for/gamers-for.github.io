## 最強武器ランキング早見表

スプラトゥーン3の最強武器ランキング。各武器の性能・サブスペ構成・環境適正を総合的に評価し、8段階のティアに分類。

### 評価基準

| 基準 | 内容 |
|------|------|
| **対象ルール** | ナワバリバトル・ガチエリア・ガチヤグラ・ガチホコ・ガチアサリの全ルール平均 |
| **重視項目** | 塗り能力・キル能力・サブスペ構成・大会使用率を総合評価 |
| **マッチ想定** | バンカラマッチ全ルール・Xマッチを想定 |
| **編成考慮** | マッチングを考慮し編成に左右されにくい武器を高評価 |

### ティア早見表

| ティア | 概要 |
|--------|------|
| **S+** | 全ルールで最強クラス。どの場面でもトップクラスの活躍が保証される |
| **S** | 環境上位。多くのルールで強く、使いこなせば勝率に大きく貢献 |
| **A+** | 安定した強さ。幅広いプレイヤーにおすすめできる優良武器 |
| **A** | 十分な戦闘力。特定の状況では上位武器にも勝てる |
| **B+〜C** | 環境的に不利だが使い手次第。愛があれば輝く武器たち |

**S-BLAST91** — 直撃でも爆風でもキルを狙える万能ブラスター。クイックボムとのコンボで確実にキルを取れ、ナイスダマの打開力も高い

**イグザミナー** — 新武器種のスピナー。ポイントセンサーで索敵しつつ、ホップソナーの波動でエリア管理ができる構成

**イグザミナーヒュー** — イグザミナーの別構成。テイオウイカの突進力で前線を押し上げる攻撃的なスタイル

**クーゲルシュライバー** — 射撃時間に応じて中距離と遠距離の両方を対応できる武器。高台から一方的に撃ち降ろせる制圧力が魅力

**シャープマーカー** — 塗りもキルもできるオールラウンダー。クイックボムのコンボが強力で、カニタンクの制圧力も非常に高い。全ルールでトップクラス

**ジムワイパーヒュー** — ポイズンミストで相手の動きを制限し、カニタンクで前線を制圧。ジムワイパーの高いキル性能をさらに活かせる構成

**スプラシューター** — スプラトゥーンを代表するオールラウンダー。3確の安定性、キューバンボムの牽制力、ウルトラショットの打開力のバランスが最高クラス

**スプラマニューバー** — スライドで回避しながら攻撃できる二丁拳銃スタイル。対面性能が非常に高く、スライド後の精度の高い射撃でキルを量産。カニタンクの制圧力も強力

**デンタルワイパーミント** — トーピードの追尾と溜め斬りのコンボが強力。ジェットパックで空中からの制圧もできる攻撃的な構成

**ノーチラス47** — チャージ保存が可能な唯一のスピナー。イカ状態でチャージを維持したまま移動でき、奇襲性能が高い

**バケットスロッシャー** — 段差を利用した攻撃が得意で、壁越しに一方的にダメージを与えられる。トリプルトルネードの打開力が高く、全ルールで安定して活躍

**ヒッセンヒュー** — 3方向にインクを飛ばすスロッシャー。エナジースタンドで味方全体を強化でき、サポート性能も高い

**プライムシューター** — 高い射程と火力を両立したシューター。3発キルの射程が長く中距離での撃ち合いに非常に強い。ラインマーカーで索敵もできる万能型

**ボトルガイザー** — 1発目の精度が非常に高く、射程も長い。タップ撃ちによる遠距離狙撃と連射の使い分けが鍵。上級者向けだがポテンシャルは最強クラス

**ボトルガイザーフォイル** — ボトルガイザーの別構成。ロボットボムで追尾しつつ、スミナガシートで視界を奪う構成
