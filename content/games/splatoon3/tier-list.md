## 評価基準

| 評価 | 意味 |
|------|------|
| **X** | 環境最強。Xマッチ上位で猛威を振るうバケモノ武器 |
| **S+** | 環境トップクラス。どのルールでも安定して強い |

| **A+** | 準強武器。練度次第で十分に戦える実力派 |
| **A** | 平均以上。特化した場面で光る一芸タイプ |
| **B+〜C** | 趣味武器。愛と練度があれば戦える |
