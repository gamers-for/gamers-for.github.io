## バトル攻略ガイド

![バナー](/images/games/splatoon3/banners/banner_008.webp)

---

### ウデマエ（ランク）の仕組み

バンカラマッチではウデマエが以下の順で上がる。

| ウデマエ | 必要条件 |
|---------|---------|
| C- → C → C+ | バンカラマッチ初参加から |
| B- → B → B+ | Cランク卒業試験をクリア |
| A- → A → A+ | Bランク卒業試験をクリア |
| S → S+ | Aランク卒業試験をクリア |
| Xマッチ解放 | 全ルールS+到達 |

---

### バトルの基本テクニック

| テクニック | 説明 |
|-----------|------|
| **イカロール** | 潜伏中に反対方向にスティックを倒してBボタン。アーマー効果あり |
| **イカノボリ** | 壁を登り切った頂上でBボタン。高所奪取に |
| **スーパージャンプ** | Xボタンでマップを開き、味方にジャンプ。前線復帰に |
| **ナイス玉** | ナイスダマ使用中に「ナイス」を押すと威力UP |
| **慣性キャンセル** | 移動方向を素早く切り替えて被弾を減らすテクニック |
