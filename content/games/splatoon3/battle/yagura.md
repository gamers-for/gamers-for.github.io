## ガチヤグラ攻略

![バナー](/images/games/splatoon3/banners/banner_010.webp)

ヤグラに乗ってレールを進め、敵陣のゴール地点まで運ぶルール。カンモン（関門）でヤグラが止まるため、チーム全体で突破する必要がある。

---

### ルール詳細

| 項目 | 内容 |
|------|------|
| **勝利条件①** | ヤグラをゴールまで運ぶ（ノックアウト） |
| **勝利条件②** | タイムアップ時にカウントが多く進んでいる方が勝利 |
| **制限時間** | 5分 |
| **ヤグラの動き** | プレイヤーが乗ると敵陣方向に進む。誰も乗っていないと中央に戻る |
| **カンモン** | ヤグラの進路に1〜3箇所のチェックポイント。一定時間ヤグラに乗り続けると突破 |
| **カンモン突破時間** | カンモンごとに異なる（約8〜12秒） |
| **延長** | 負けているチームがヤグラに乗っていれば延長 |

### カンモンの仕組み

- ヤグラがカンモンに到達すると**一時停止**する
- プレイヤーがヤグラに乗り続けると**カンモンゲージ**が進行
- ゲージが満タンになるとカンモン突破
- 乗り手がやられると**ゲージは途中で止まる**（リセットはされない）
- 複数人で乗ってもゲージの進行速度は変わらない

---

### 攻めの立ち回り

#### ヤグラに乗る人のコツ

| コツ | 詳細 |
|------|------|
| **ヤグラの柱を盾にする** | ヤグラの柱の裏に隠れて被弾を減らす |
| **塗りで自衛する** | ヤグラ周辺をメインウェポンで塗りつつ乗り続ける |
| **イカセンプクで回避** | ヤグラ上で潜伏して敵のインクを避ける |
| **やられそうなら降りる** | 無理に乗り続けるよりも生存してすぐ再乗車 |

#### 護衛する人のコツ

| コツ | 詳細 |
|------|------|
| **ヤグラの前を制圧** | ヤグラより前に出て、敵が近づけないようにする |
| **高台を取る** | ヤグラ周辺の高台から敵を牽制 |
| **ヤグラに乗らない** | 護衛が全員乗ると、敵に囲まれて全滅する |
| **交代の意識** | 乗り手がやられたら即座にヤグラに乗る |

#### カンモン突破の方法

1. カンモン手前でスペシャルを溜めておく
2. カンモン到達と同時にスペシャル発動（2人以上で合わせる）
3. 敵を排除or退かせた隙にカンモンゲージを進める
4. グレートバリアでヤグラ上を守るのが非常に有効

### 守りの立ち回り

| 状況 | 対処法 |
|------|--------|
| **ヤグラ接近時** | ヤグラに乗っている敵をキルする。ボムを投げ込むのが有効 |
| **カンモン防衛** | カンモンで止まっている間に全力でキル。ボムやスペシャルをヤグラに集中 |
| **ヤグラ上の敵が多い時** | 無理に近づかず、長射程武器で牽制 |
| **味方がデスした時** | 1人で無理に止めず、復帰を待つ |

### 武器種別の役割

| 武器種 | 役割 |
|--------|------|
| **ブラスター** | ヤグラ上の敵に爆風が当たりやすい。ヤグラ乗りにも護衛にも◎ |
| **スロッシャー** | ヤグラ上から攻撃しやすい。バケツ系は乗り手に向いている |
| **チャージャー** | 護衛の後衛。敵がヤグラに近づくのを阻止 |
| **ローラー** | カンモン付近で潜伏キル。敵の注意がヤグラに向いている隙に奇襲 |
