## Xマッチ攻略

Xマッチはバンカラマッチで最高ランク「S+」に到達後に挑戦できる上級者向けモード。Xパワーという独自のレーティングで実力が数値化される。

---

### Xマッチの仕組み

| 項目 | 内容 |
|------|------|
| **解放条件** | バンカラマッチの全4ルールでS+に到達 |
| **Xパワー** | 実力を示す数値。勝つと上がり、負けると下がる |
| **初期計測** | シーズン開始時に計測マッチ（10戦）を行い初期Xパワーが決定 |
| **マッチング** | 近いXパワーの8人でマッチング |
| **ルール** | ガチエリア、ガチヤグラ、ガチホコ、ガチアサリの4ルール |
| **シーズン** | 約3ヶ月ごとにシーズンが切り替わりXパワーがリセット |
| **Xランキング** | シーズン中のXパワー上位500人がランキングに掲載 |

### Xパワーの変動

| 条件 | パワー変動 |
|------|----------|
| 勝利 | +8〜+15程度（相手との差による） |
| 敗北 | -8〜-15程度（相手との差による） |
| 格上に勝利 | 大きく上昇 |
| 格下に敗北 | 大きく下降 |
| 連勝 | 加速ボーナスあり |

### Xパワーの目安

| Xパワー | レベル感 | 人口割合（目安） |
|---------|---------|--------------|
| 1500〜1800 | Xマッチ入門。S+から上がったばかり | 多い |
| 1800〜2000 | 中級Xプレイヤー。安定した腕前 | 中程度 |
| 2000〜2200 | 上級者。ルール理解と武器練度が高い | やや少ない |
| 2200〜2500 | 最上位層。全国トッププレイヤー | 非常に少ない |
| 2500以上 | 最強クラス。Xランキング常連 | 極めて少ない |

---

### パワー帯別のステップアップガイド

#### 1500〜1800帯の壁を超えるには

- **デス数を減らす** — 1デスの影響がバンカラマッチより大きい。無理な突撃を控え、人数不利では退く判断を
- **スペシャルを無駄遣いしない** — 打開タイミングで味方と合わせて使う意識を持つ
- **マップをこまめに確認** — Xボタンで味方・塗り状況を把握。裏取り警戒も重要

#### 1800〜2000帯の壁を超えるには

- **武器を1〜2本に絞る** — 練度が高い武器で戦う方がパワーが安定する
- **ステージのポジションを覚える** — 各ステージの強ポジション、射線の通し方を把握
- **状況判断のスピードを上げる** — 「今攻めるか退くか」を瞬時に判断する力

#### 2000以上を目指すには

- **勝ちパターンを持つ** — 自分の武器で確実にキルが取れるパターンを複数持つ
- **味方の動きに合わせる** — VC（ボイスチャット）がなくてもマップで味方の動きを読む
- **相手の武器構成を見て対応を変える** — 試合開始時に相手武器を確認し、立ち回りを調整
