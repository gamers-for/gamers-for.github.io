## Xマッチ攻略

Xマッチはバンカラマッチで最高ランク「S+」に到達後に挑戦できる上級者向けモード。

---

### Xマッチの仕組み

| 項目 | 内容 |
|------|------|
| **解放条件** | バンカラマッチ全ルールでS+到達 |
| **Xパワー** | 実力を示す数値。勝つと上がり、負けると下がる |
| **初期Xパワー** | 計測マッチ（10戦）の結果で決定 |
| **マッチング** | 近いXパワーの8人でマッチング |
| **ルール** | バンカラマッチと同じ4ルール（エリア、ヤグラ、ホコ、アサリ） |
| **シーズンリセット** | シーズン切替時にXパワーがリセット |

### Xパワーの目安

| Xパワー | レベル感 |
|---------|---------|
| 1500〜1800 | Xマッチ入門。S+レベル |
| 1800〜2000 | 中級Xプレイヤー |
| 2000〜2200 | 上級者。安定して高パワーを維持 |
| 2200〜2500 | 最上位層。トッププレイヤー |
| 2500以上 | 日本トップクラス |

### Xマッチで勝つために

1. **武器を絞る**: メイン武器1〜2本に絞って練度を上げる
2. **スペシャルの合わせ**: 味方と連携してスペシャルを同時発動
3. **デスを減らす**: 1デスの重みが大きい。無理な突撃を避ける
4. **マップ管理**: 常にXボタンでマップを確認し、状況判断する
5. **ステージの理解**: 各ステージのポジション・ルートを完全に把握する
