## ガチホコ攻略

![バナー](/images/games/splatoon3/banners/banner_011.webp)

ガチホコを持って敵のゴール地点まで運ぶルール。ホコを持つ人と護衛する人の連携が勝敗を分ける。

---

### ルール詳細

| 項目 | 内容 |
|------|------|
| **勝利条件①** | ガチホコを敵のゴール台座に置く（ノックアウト） |
| **勝利条件②** | タイムアップ時にカウントが多く進んでいる方が勝利 |
| **制限時間** | 5分 |
| **ホコバリア** | 試合開始時、中央にホコバリアで包まれたガチホコが出現 |
| **バリア破壊** | ホコバリアを攻撃して割る。割れた瞬間に爆発（範囲ダメージ） |
| **ホコ持ち制限** | ホコを60カウント持ち続けると自爆する |
| **ホコショット** | ホコを持つとメインウェポンの代わりにホコショット（チャージ式長射程）が使える |
| **延長条件** | 負けているチームがホコを持っている、またはホコが落ちている状態で延長 |

### ホコバリアの仕組み

- バリアは**塗り攻撃で破壊可能**（ダメージ量が蓄積）
- 破壊時に**大爆発**が発生 → 近くの敵を倒せる / 味方もダメージを受ける
- 爆発のインクは**割ったチームの色**になる
- **先に割った方が有利**: ホコ取得の主導権を握れる

---

### ガチホコの特殊ルール

| ルール | 詳細 |
|--------|------|
| **60カウント自爆** | ホコを60秒持つと自爆。持ちっぱなしは不可 |
| **スペシャル自動チャージ** | ホコを持っていないチームのスペシャルが少しずつ溜まる |
| **延長戦の条件** | 負けチームがホコ持ち or ホコが落ちている状態なら延長。勝ちチームがホコを確保すると即終了 |
| **ホコが海に落ちた時** | 一定時間後にバリア付きで中央にリセット |

### ホコ割りのコツ

| コツ | 詳細 |
|------|------|
| **全員で割る** | 4人で集中攻撃が最速。ボムも有効 |
| **爆発を利用** | バリア破壊の爆発で敵を倒せるので、敵が近い時に割ると有利 |
| **割られる側の対処** | バリア破壊時は距離を取るか、壁の裏に隠れる |

### ホコ持ちの立ち回り

| 状況 | 行動 |
|------|------|
| **味方が前にいる時** | 積極的に進む。味方の塗りに沿って前進 |
| **味方がいない時** | 無理に進まない。安全な場所でホコショットで援護 |
| **敵と対面した時** | ホコショットで遠距離攻撃。接近されたら味方に任せる |
| **やられそうな時** | できるだけ前進してからやられる（カウントを1でも進める） |

### ルート選択

多くのステージには**メインルート**と**サブルート**がある。

| ルート | メリット | デメリット |
|--------|---------|-----------|
| **メインルート** | 道幅が広く塗りやすい、味方が護衛しやすい | 敵に読まれやすい |
| **サブルート** | 意外性があり敵の守りが薄い | 道が狭い、護衛が分散する |

- 基本は**メインルート**で攻める
- 何度もメインルートが止められたら**サブルート**に切り替え
- ステージごとのルートを覚えることが上達の鍵
