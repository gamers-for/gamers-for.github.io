## ガチアサリ攻略

![バナー](/images/games/splatoon3/banners/banner_012.webp)

マップ上のアサリを10個集めてガチアサリを作り、敵のゴールに投げ入れるルール。バンカラマッチの中で最も複雑なルールだが、奥が深い。

---

### ルール詳細

| 項目 | 内容 |
|------|------|
| **勝利条件①** | 相手のカウントを先に0にする（ノックアウト） |
| **勝利条件②** | タイムアップ時にカウントが少ない方が勝利 |
| **制限時間** | 5分 |
| **アサリの入手** | マップ上に散らばっているアサリを拾う |
| **ガチアサリ作成** | アサリを10個集めるとガチアサリに変化 |
| **ゴールバリア** | 敵ゴールにはバリアがある。ガチアサリを投げ込むと破壊 |
| **チャンスタイム** | バリア破壊後、一定時間通常アサリを投げ込める |
| **アサリのカウント** | ガチアサリ＝20p、通常アサリ＝3p |

### アサリの特殊ルール

| ルール | 詳細 |
|--------|------|
| **アサリ所持上限** | 1人最大10個（=ガチアサリ1個分） |
| **アサリの再出現** | 拾われたアサリは一定時間後に別の場所に再出現 |
| **デス時のアサリ** | やられると持っていたアサリをその場にばらまく。敵に奪われる |
| **ガチアサリ作成時** | 味方全員のスペシャルゲージが少し溜まる |
| **延長条件** | 負けチームがガチアサリを持っている or チャンスタイム中なら延長 |

---

### ゴール攻めの方法

#### ステップ1: ガチアサリの準備

- アサリを9個まで集めておく（10個でガチアサリになると光って目立つ）
- 味方に「カモン」でアサリを投げてもらい、10個にする
- ゴール付近に味方を集結させてから10個にする

#### ステップ2: バリア破壊

- ガチアサリを敵ゴールのバリアに投げ込む
- バリアが破壊され、**チャンスタイム**開始
- この瞬間に味方全員がアサリを投げ込む

#### ステップ3: チャンスタイム

| 行動 | ポイント |
|------|---------|
| 持っているアサリを全て投げ込む | 通常アサリ1個＝3p |
| 味方とアサリの受け渡し | ZRでアサリを投げて受け渡し可能 |
| 敵のキルを取って邪魔されないようにする | ゴール付近で護衛する人が必要 |

### 防衛の方法

| 状況 | 対処法 |
|------|--------|
| **敵がガチアサリを持っている** | ガチアサリ持ちを最優先でキル。ガチアサリは光って目立つ |
| **チャンスタイム中** | ゴール付近で敵を倒す。アサリを投げ込ませない |
| **チャンスタイム終了後** | バリアが復活するまで耐える |
| **自陣にアサリが多い** | アサリを拾って敵に使わせない（妨害目的） |

### 武器別の役割

| 役割 | 向いている武器種 | 仕事 |
|------|----------------|------|
| **アサリ集め＋ゴール攻め** | 短射程シューター、マニューバー | 機動力を活かしてアサリを集め、ゴールに投げ込む |
| **ルート確保** | 中射程シューター、ローラー | ゴールまでの道を塗って確保 |
| **護衛** | チャージャー、スピナー | ゴール付近で味方を守り、敵を排除 |
| **妨害** | ブラスター、スロッシャー | 敵のアサリ集めを妨害。敵を倒してアサリを奪う |

### おすすめギアパワー

| ギアパワー | 理由 |
|-----------|------|
| **イカダッシュ速度アップ** | アサリ集めの効率UP、ゴールへの到達速度UP |
| **スペシャル増加量アップ** | ゴール攻めのきっかけを作るスペシャル回転率UP |
| **ステルスジャンプ** | 味方のゴール攻めに合わせて前線にジャンプ |
