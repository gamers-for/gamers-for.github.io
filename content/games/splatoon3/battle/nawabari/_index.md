## ナワバリバトル攻略

![バナー](/images/games/splatoon3/banners/banner_008.webp)

3分間で自チームの色で多く塗ったチームが勝利するルール。スプラトゥーン3の基本ルールであり、全プレイヤーが最初にプレイするモード。

---

### ルール詳細

| 項目 | 内容 |
|------|------|
| **制限時間** | 3分 |
| **勝利条件** | タイムアップ時に塗り面積が多いチームの勝利 |
| **人数** | 4対4 |
| **ステージ** | 2時間ごとにローテーション（2ステージ） |
| **ウデマエ** | なし（レギュラーマッチ）。フェス時はフェスマッチに変化 |
| **塗り判定** | 塗れない場所（壁の上部、柵、一部オブジェクト）は判定外 |

### 塗りの仕組み

- **床のみがカウント対象**: 壁はカウントされない（移動には重要）
- **上書き可能**: 敵が塗った場所を自分のインクで塗り返せる
- **塗れない場所**: ステージの端、一部のオブジェクト上面
- **スタート地点周辺**: リスポーン地点は常に自チームの色

---

### 時間帯別の立ち回り

#### 序盤（残り3:00〜2:00）

| やるべきこと | やってはいけないこと |
|------------|-------------------|
| 自陣の安全な場所を塗って前進 | 自陣を塗りすぎて中央に遅れる |
| 中央付近まで到達して陣地を確保 | 1人で敵陣に突っ込む |
| サブウェポンで牽制しつつ塗る | 味方と固まりすぎて塗り効率を下げる |

#### 中盤（残り2:00〜0:30）

| やるべきこと | やってはいけないこと |
|------------|-------------------|
| 中央エリアの塗りを維持 | 無理にキルを狙って前に出すぎる |
| スペシャルを溜めてラストに備える | スペシャルを無駄遣いする |
| 敵が攻めてきたら引きながら塗る | やられた後にすぐ突っ込んで連続デス |

#### 終盤（残り0:30〜0:00）— 最重要

| やるべきこと | やってはいけないこと |
|------------|-------------------|
| **自陣の塗り残しを全て埋める** | 敵陣に攻め込んでデスする |
| **スペシャルを全て使い切る** | スペシャルを温存したまま終了 |
| 生存を最優先にする | 残り10秒で前線に飛び出す |
| 味方の塗りと被らない場所を塗る | 味方が塗っている場所を重複して塗る |

### おすすめ武器種

#### 塗り重視（初心者おすすめ）

| 武器種 | おすすめ理由 |
|--------|------------|
| **シューター** | 塗り効率が高く、キルも安定。[わかばシューター](../../weapons/わかばシューター/)が特におすすめ |
| **ローラー** | 転がすだけで広範囲を一気に塗れる |
| **フデ** | ZR押しっぱなしで超高速塗り。移動しながら塗れる |

#### キル重視（上級者向け）

| 武器種 | おすすめ理由 |
|--------|------------|
| **チャージャー** | 敵を倒して人数有利を作り、塗らせない |
| **マニューバー** | スライドで機動力が高く、撃ち合いに強い |
| **ブラスター** | 爆風で複数の敵を動かせる |

### 塗りで勝つためのテクニック

1. **塗り返しよりも新しい塗り**: 敵が塗った場所を塗り返すより、誰も塗っていない場所を塗る方が効率的
2. **壁を塗る**: カウントには含まれないが、移動ルートとして重要。壁を塗ると奇襲ルートが生まれる
3. **サブウェポンで遠くを塗る**: ボム系サブは遠くの塗りに有効
4. **マップを見る習慣**: X ボタンでマップを開き、塗り残しを確認

### スペシャルの使いどころ

| タイミング | おすすめスペシャル |
|-----------|-----------------|
| 中央制圧 | トリプルトルネード、アメフラシ |
| ラスト30秒の塗り | ナイスダマ、ウルトラショット |
| 敵の排除 | カニタンク、ジェットパック |
| 防衛 | グレートバリア、ホップソナー |
