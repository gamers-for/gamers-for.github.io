## ガチエリア攻略

![バナー](/images/games/splatoon3/banners/banner_009.webp)

ガチエリアは指定エリアを塗って確保し、カウントを先に0にしたチームが勝利するルール。バンカラマッチの中でも特にチームワークが問われるルール。

---

### ルール詳細

| 項目 | 内容 |
|------|------|
| **勝利条件① ** | 相手のカウントを先に0にする（ノックアウト） |
| **勝利条件②** | タイムアップ時にカウントが少ない方が勝利 |
| **制限時間** | 5分 |
| **エリア数** | ステージにより1〜2箇所 |
| **確保条件** | エリア内の一定割合を自チームの色で塗る |
| **カウント進行** | 確保状態を維持している間、カウントが減っていく |
| **ペナルティ** | エリアを取り返された後、再確保するとペナルティカウントが追加 |
| **延長** | タイムアップ時に負けているチームがエリアを確保していれば延長 |

### 「確保」と「カウント」の仕組み

- エリア内の約70%以上を自チームの色で塗ると「確保」状態になる
- 確保中はカウントが1ずつ減っていく（100→99→...→0）
- 敵に奪われると「ペナルティカウント」が追加される
- ペナルティを消化してから本カウントが再開

---

### 打開のセオリー（攻め）

エリアを取られている状態から奪い返す方法。

| ステップ | 行動 | ポイント |
|---------|------|---------|
| ① 溜める | スペシャルゲージを全員で溜める | 復帰後すぐに突っ込まない。リスポーン付近で塗ってスペシャルを溜める |
| ② 合わせる | スペシャルを2〜3人同時に発動 | 「カモン」シグナルでタイミングを合わせる |
| ③ 塗り返す | エリアに大量のインクを投入 | トリプルトルネード、アメフラシ等でエリアを一気に塗る |
| ④ キルを取る | 人数有利を作る | 1〜2キルで人数有利を確保 |
| ⑤ 確保 | エリアを塗り切って確保 | 確保後はすぐに防衛の立ち位置へ移行 |

### 防衛のセオリー（守り）

エリアを確保した後、守り続ける方法。

| ステップ | 行動 | ポイント |
|---------|------|---------|
| ① 高台を取る | エリアを見下ろせる高台に陣取る | 射程の長い武器が特に有効 |
| ② 前線を押す | エリアより敵側に味方を2人配置 | 前線で敵を足止めし、エリアに近づけさせない |
| ③ エリアを塗る | 残り2人でエリアの塗りを維持 | 少しでも敵インクが入ったらすぐに塗り返す |
| ④ 退く判断 | 敵のスペシャルが来たら退く | 無理に防衛してデスするより、退いて打開に切り替える |

### 武器の役割分担

| 役割 | 担当武器種 | 仕事 |
|------|-----------|------|
| **前衛キル** | 短射程シューター、マニューバー | 敵を倒して人数有利を作る |
| **塗りキープ** | 中射程シューター、ローラー | エリアの塗りを維持する |
| **後衛サポート** | チャージャー、スピナー | 高台から敵を牽制・キル |
| **スペシャル打開** | 塗り武器全般 | スペシャルを高速回転して打開を仕掛ける |

### おすすめギアパワー

| ギアパワー | 理由 |
|-----------|------|
| **スペシャル増加量アップ** | 打開に必要なスペシャルを高速で溜める |
| **スペシャル減少量ダウン** | デス時のスペシャルロスを軽減 |
| **カムバック** | 復活後のスペシャル溜め速度UP |
