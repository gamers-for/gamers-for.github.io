## サイドオーダー攻略

![バナー](/images/games/splatoon3/banners/banner_005.webp)

サイドオーダーは有料DLC「エキスパンション・パス」第2弾に含まれるローグライク型の1人用モード。秩序の塔を登り、最上階のフロア30を目指す。主人公は「ハチ」（オクトエキスパンションの主人公）。

### 基本情報

| 項目 | 内容 |
|------|------|
| **ジャンル** | ローグライクアクション |
| **プレイ人数** | 1人 |
| **前提条件** | エキスパンション・パス購入 |
| **主人公** | ハチ（8号） |
| **舞台** | 秩序の塔 |
| **最終フロア** | 30F |
| **敵** | スケルトーン（秩序側の敵） |

### コアシステム解説

#### パレット

パレットは使用するブキとベースカラーのセット。パレットごとにブキが決まっており、カラーチップで強化していく。初回はシューターパレットがおすすめ。

#### カラーチップ

各フロアクリア後にカラーチップを選択できる。チップには色があり、同じ色を集めるとボーナス効果が発動する。チップの選択がビルドの核になる。

| チップ効果 | 内容 |
|-----------|------|
| **攻撃力アップ** | メインウェポンの火力を上昇 |
| **インク効率アップ** | インク消費量を軽減 |
| **移動速度アップ** | イカダッシュやヒト移動の速度を上昇 |
| **回復系** | ライフ回復や被ダメージ軽減 |
| **スペシャル系** | スペシャルの威力やゲージ増加 |

#### シンジュ（真珠）

シンジュはサイドオーダーの強化通貨。塔の攻略で獲得でき、パレットの永続強化に使用できる。

| 使い道 | 効果 |
|--------|------|
| **パレット強化** | 各パレットの基礎能力をアップ |
| **新パレット解放** | 新しいブキのパレットを使用可能に |
| **ハッキング** | 秩序の塔の情報を解放 |

**おすすめ使い道**: まずは使用するパレットの基礎強化に投資するのが効率的。

#### ネリコイン

ネリコインは塔の中で使える通貨。自販機でカラーチップやアイテムを購入できる。

#### ハッキング

塔を登るたびにハッキングが進行し、ストーリーが解放される。イイダの開発日記など、秩序の塔の謎が徐々に明らかになる。

### フロアの種類

| フロアタイプ | 内容 |
|------------|------|
| **通常フロア** | スケルトーンを全滅させてクリア |
| **ボスフロア** | 10F、20F、30Fに出現する強力なボス |
| **ショップフロア** | 自販機でカラーチップやアイテムを購入 |
| **イベントフロア** | 特殊なミッション（時間制限、特定条件クリアなど） |

### ボス攻略

| フロア | ボス | 攻略のコツ |
|--------|------|-----------|
| **10F** | 秩序の守護者 | 攻撃パターンを覚えて回避。攻撃チップを多めに取っておく |
| **20F** | 秩序の支配者 | 複数の攻撃パターンに対応。回復系チップがあると安心 |
| **30F** | 最終ボス | 全フェーズの攻撃を回避しつつ弱点を攻撃。ビルドが完成していれば問題なし |

### 効率的な攻略法

1. **攻撃系チップを序盤で確保** — 最初の5フロアは火力アップのチップを優先して取る
2. **同色チップを意識的に集める** — 同じ色を集めるとセットボーナスが発動し大幅強化
3. **自販機は必要なものだけ購入** — ネリコインは有限。本当に必要なチップだけ買う
4. **10F前に攻撃ビルドを固める** — 最初のボス戦に備えて火力を確保
5. **回復系チップは中盤から** — 20F以降は敵が強いので生存力も必要になる
6. **リトライを恐れない** — 失敗しても獲得したシンジュやハッキングは保持される

### クリア後の報酬

| 報酬 | 内容 |
|------|------|
| **レプリカ武器** | 各パレットでクリアするとレプリカ武器が解放され、オンラインで使用可能 |
| **ロッカーアイテム** | サイドオーダー限定のロッカー装飾品 |
| **バッジ** | クリア回数や特定条件で限定バッジを獲得 |
| **ストーリー解放** | ハッキングの進行でイイダや秩序の塔の物語が解放される |
| **ハチのパレット** | 全パレットクリア後に解放される特別パレット |

### クリアできない時の対処法

- シンジュでパレットの基礎能力を強化する（永続効果）
- シューターパレットなど扱いやすいパレットで練習する
- 攻撃チップを重点的に集めるビルドに変更する
- 同色チップのセットボーナスを狙って効率的に強化
- 自販機で不足しているチップを補充する
