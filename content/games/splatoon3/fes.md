## フェス攻略・過去フェス結果まとめ

![バナー](/images/games/splatoon3/banners/banner_006.webp)

フェスはスプラトゥーン3の定期開催イベント。3つの陣営に分かれて期間限定のバトルを楽しむ。勝利チームには追加のスーパサザエが報酬として贈られる。

### フェスの流れ

| フェーズ | 内容 |
|---------|------|
| **ヨビ祭** | フェス開始前の投票期間。広場でお題を選択して陣営に参加 |
| **前半戦（ナワバリ）** | 通常のナワバリバトル（4vs4）でフェスポイントを稼ぐ |
| **中間発表** | 前半戦の得票率・バトル勝率・塗りポイントの途中経過が発表される |
| **後半戦（トリカラ追加）** | トリカラバトル（4vs2vs2）が追加。中間1位チームが防衛側に |
| **結果発表** | 最終結果と報酬の受取。スーパサザエの量が勝利チームほど多い |

### フェスで遊べるルール

#### ナワバリバトル（4vs4）

フェス期間中のメインルール。通常のナワバリバトルと同じく3分間で塗り面積を競う。フェスTを着用してプレイする。

#### トリカラバトル（4vs2vs2）

後半戦で解放される特殊ルール。中間発表で1位のチーム（4人）が防衛側、2位・3位チーム（各2人）が攻撃側として戦う。

- **攻撃側（2人チーム×2）**: 中央のスーパーシグナルを奪うことが最優先。奪取すると一定時間そのチームの色にエリアが塗られる
- **防御側（4人チーム）**: 自陣のスーパーシグナルを守りつつ、2チームが争う隙を突く
- **立ち回りのコツ**: 攻撃側は味方と連携しシグナルに素早くアクセス。防御側は2方向からの攻撃に対処する必要がある

#### 10倍マッチ / 100倍マッチ / 333倍マッチ

フェス中にランダムで発生する倍率マッチ。獲得フェスポイントが通常の10倍/100倍/333倍になるスペシャルバトル。333倍マッチは非常にレアで、マッチング時に特別な演出が入る。

### フェスパワーの仕組み

| 項目 | 内容 |
|------|------|
| **初期パワー** | 前回のXパワーや過去の戦績に基づいて決定 |
| **変動** | 勝つとパワーが上がり、負けると下がる |
| **貢献度** | フェスパワーが高いほどチームへの貢献度が大きい |
| **確認方法** | ロビー端末やスマホアプリからフェスパワーを確認可能 |

### フェス称号（ランク）一覧

フェスではバトルを重ねることでフェス称号がランクアップする。

| 称号 | 到達条件 |
|------|---------|
| **フェスのでし** | 初期ランク |
| **まとい** | フェスポイントを一定数獲得 |
| **カリスマ** | さらにフェスポイントを獲得 |
| **えいえんのフェス王/女王** | 最高ランク。大量のフェスポイントが必要 |

### フェスT（Tシャツ）の仕組み

- フェスTはフェス参加時に支給される特別なギア
- クリーニングすることでギアパワーを付け替え可能
- フェスTの入手はフェス開始時に自動で行われる
- フェス終了後は通常のギアには戻るが、次回フェスで再度支給される

### フェス報酬

| 報酬 | 入手条件 |
|------|---------|
| **スーパサザエ** | フェス参加。勝利チームは追加サザエがもらえる |
| **フェスポイント報酬** | 一定ポイントごとにカプセルマシン回数やアイテムを獲得 |
| **限定称号** | 各フェスのお題に応じた限定称号 |
| **王冠** | 前回フェスの最高フェスパワー上位プレイヤーに付与される特別マーク |

### フェスの攻略ポイント

1. **フェスTのギアパワーを整える** — フェスT専用のギアパワーを事前に付けておく
2. **10倍/100倍マッチは全力で** — 倍率マッチは勝敗が大きく影響するため全力で臨む
3. **後半戦のトリカラに挑戦** — フェスの醍醐味であるトリカラバトルを楽しもう
4. **フレンドとグループマッチ** — フレンドと一緒にプレイすると勝率アップ
