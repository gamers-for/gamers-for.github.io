## ギアパワー最強ランキング

### Xランク（最強・必須級）

| ギアパワー | 効果 | おすすめ理由 |
|-----------|------|------------|
| イカダッシュ速度アップ | イカダッシュ時の移動速度UP | 機動力向上、全武器で汎用性トップ |
| ステルスジャンプ | スーパージャンプの着地マーカーを遠距離から隠す | 長射程からの着地狩り対策に必須 |
| カムバック | 復活後20秒間、複数ステータス向上 | デス後の復帰が強力。前衛武器に最適 |
| スペシャル減少量ダウン | やられた時のスペシャルゲージ減少量ダウン | デスが多くてもスペシャルを維持 |
| スーパージャンプ時間短縮 | スーパージャンプの時間短縮 | 前線復帰を大幅に高速化 |
| 相手インク影響軽減 | 敵インク上のダメージ・速度低下を軽減 | 前線での生存力が大幅UP |
| インク回復力アップ | インクタンクの回復速度UP | 効果が実感しやすく汎用性高い |

| ギアパワー | 効果 |

| インク効率アップ（メイン） | メインウェポンの消費インク軽減 |
| イカニンジャ | イカダッシュのインク飛沫を隠す（速度低下あり） |
| アクション強化 | イカロール・イカノボリ性能向上、ジャンプ撃ちのブレ軽減 |
| スペシャル増加量アップ | スペシャルゲージの増加量UP |
| 復活時間短縮 | 連続でやられた時の復活時間短縮 |
| ラストスパート | 残り30秒でインク効率大幅UP |
| 復活ペナルティアップ | 倒した相手の復活時間＋スペシャル減少を増加 |
