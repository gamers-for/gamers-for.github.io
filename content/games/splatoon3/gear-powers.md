## ギアパワー一覧

![バナー](/gamers-for/images/games/splatoon3/banners/banner_001.webp)

ギアパワーは装備するギア（アタマ・フク・クツ）に付いている特殊効果。正しい組み合わせで戦闘力が大きく変わる。

### 基本ギアパワー

| ギアパワー | 効果 | おすすめ武器種 |
|-----------|------|--------------|
| **インク効率アップ(メイン)** | メインウェポンのインク消費を軽減 | 全武器種 |
| **インク効率アップ(サブ)** | サブウェポンのインク消費を軽減 | ボム系サブ持ち |
| **インク回復力アップ** | インクの回復速度を上昇 | インク消費の激しい武器 |
| **ヒト移動速度アップ** | ヒト状態での移動速度を上昇 | スピナー、スロッシャー |
| **イカダッシュ速度アップ** | イカダッシュの移動速度を上昇 | 全武器種（汎用的） |
| **スペシャル増加量アップ** | スペシャルゲージの増加量をアップ | 強スペシャル持ち |
| **スペシャル減少量ダウン** | やられた時のスペシャルゲージ減少を抑える | 前線武器 |
| **スペシャル性能アップ** | スペシャルの効果時間や範囲を強化 | ジェットパック等 |
| **復活時間短縮** | やられてからの復活が早くなる | 前線武器、ローラー |
| **スーパージャンプ時間短縮** | スーパージャンプが素早くなる | 全武器種 |
| **サブ影響軽減** | 相手のサブウェポンの影響を軽減 | 全武器種（汎用的） |
| **相手インク影響軽減** | 相手のインクを踏んだ際のペナルティ軽減 | 前線武器 |
| **アクション強化** | ジャンプ撃ちの精度を向上 | シューター、マニューバー |
| **サブ性能アップ** | サブウェポンの性能を強化 | ボム飛距離アップ等 |

### ギア構成の基本

1. **メイン性能特化型** — インク効率(メイン)を積んでメインでの戦闘力を重視
2. **スペシャル回転型** — スペシャル増加量をメインに積んでスペシャル連発
3. **機動力特化型** — 移動速度系を積んでポジショニングで勝つ
4. **バランス型** — 各種ギアを少しずつ積んで万能に
