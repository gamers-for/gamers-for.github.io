スプラトゥーン3の**全26種ギアパワー**の効果と評価をまとめています。ギアパワーの優先順位がわからない方は、このランキングを参考にしてください。

## ギアパワーおすすめランキング

### 基本ギアパワー（14種）

基本ギアパワーは**アタマ・フク・クツすべてに付く**汎用的なギアパワーです。

| ランク | ギアパワー名 | 効果 |
|--------|-------------|------|
| **X** | **インク回復力アップ** | インクタンク回復速度がアップ |
| **X** | **イカダッシュ速度アップ** | イカダッシュ時の移動速度がアップ |
| **X** | **スペシャル減少量ダウン** | やられた時のスペシャルゲージ減少量がダウン |
| **X** | **スーパージャンプ時間短縮** | スーパージャンプにかかる時間が短縮 |
| **X** | **相手インク影響軽減** | 相手インクを踏んだ時のダメージと速度低下を軽減 |
| **S+** | **インク効率アップ（メイン）** | メインウェポンの消費インク量が少なくなる |
| **S+** | **インク効率アップ（サブ）** | サブウェポンの消費インク量が少なくなる |
| **S+** | **ヒト移動速度アップ** | ヒト状態の移動速度がアップ |
| **S+** | **スペシャル増加量アップ** | スペシャルゲージの増加量がアップ |
| **S+** | **復活時間短縮** | 連続でやられた時の復活時間が短縮 |
| **S+** | **アクション強化** | イカロール・イカノボリの性能向上、ジャンプ撃ちのブレ軽減 |
| **A** | **スペシャル性能アップ** | スペシャルウェポンの性能がアップ |
| **A** | **サブ性能アップ** | サブウェポンの性能がアップ（飛距離など） |
| **A** | **サブ影響軽減** | サブウェポンの影響を軽減 |

### メイン専用ギアパワー（12種）

メイン専用ギアパワーは**アタマ・フク・クツのメインスロットにのみ付く**特殊なギアパワーです。

| ランク | ギアパワー名 | 効果 |
|--------|-------------|------|
| **X** | **カムバック** | 復活後20秒間、複数のステータスが向上 |
| **X** | **ステルスジャンプ** | スーパージャンプの着地マーカーを遠距離から見えなくする |
| **S+** | **ラストスパート** | 試合残り30秒、インク効率がアップ |
| **S+** | **イカニンジャ** | イカダッシュ時のインク飛沫を隠す（速度は低下） |
| **S+** | **復活ペナルティアップ** | 自分が倒した相手の復活時間とスペシャル減少を増加 |
| **A** | **サーマルインク** | メインの弾を当てた相手を一定時間表示 |
| **A** | **追加ギアパワー倍化** | 追加ギアパワーの効果を倍にする（1つ分） |
| **A** | **対物攻撃力アップ** | プレイヤー以外のオブジェクトへのダメージが増加 |
| **A** | **受け身術** | スーパージャンプ着地時に前転して隙を減らす |
| **B** | **スタートダッシュ** | 試合開始後30秒間、移動速度アップ |
| **B** | **逆境強化** | 味方が少ない時、スペシャルゲージが徐々に増加 |
| **B** | **リベンジ** | 自分を倒した相手を一定時間マーキング |

| クツ | 対物攻撃力アップ | イカダッシュ速度アップ×3 |

| 部位 | メイン | サブ×3 |
|------|--------|--------|
| アタマ | カムバック | スペシャル増加量アップ×3 |

| クツ | ステルスジャンプ | インク回復力アップ×3 |

## ギアパワーの仕組み

- 各装備に**メインスロット1つ + サブスロット3つ**（合計12スロット）
- メインスロット = サブスロット×3.3倍の効果
- **57表記**: メイン=10, サブ=3 で合計57が最大
- ほとんどのギアパワーは**効果が減衰する**（積みすぎ注意）
- メイン専用ギアパワーは**メインスロットにしか付かない**ため最大3つまで
