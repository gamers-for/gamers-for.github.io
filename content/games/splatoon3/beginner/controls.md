## 操作方法・おすすめ感度設定

![バナー](/images/games/splatoon3/banners/banner_001.webp)

スプラトゥーン3の基本操作とおすすめ設定を解説。ジャイロ操作は上達に必須級なので、最初から慣れておくことを推奨。

---

### 基本ボタン操作

| ボタン | 操作内容 |
|--------|---------|
| **Lスティック** | 移動（倒し具合で速度が変化） |
| **Rスティック** | 視点移動（左右の大きな視点変更） |
| **ZR** | メインウェポンで攻撃 |
| **ZL** | イカセンプク（インクに潜る） |
| **R** | サブウェポンを使用 |
| **Rスティック押し込み** | スペシャルウェポン発動 |
| **B** | ジャンプ |
| **Y** | 視点リセット（正面に戻す） |
| **X** | マップを開く（味方にスーパージャンプ可能） |
| **十字キー↑** | 「カモン！」（やられ中は「やられた！」） |
| **十字キー↓** | 「ナイス！」 |
| **Aボタン（マップ中）** | 選択した味方にスーパージャンプ |
| **-ボタン** | ロビーのメニューを開く |

### イカセンプクの重要性

| 状態 | 効果 |
|------|------|
| **ZL長押し** | 自チームのインクに潜る。移動速度UP＋インク回復 |
| **潜伏中** | 敵から見えなくなる（相手インク上では不可） |
| **イカロール** | 潜伏→反対方向に急速移動。アクション強化で強化可能 |
| **イカノボリ** | 壁を塗って登った後、頂上でジャンプ。高所奪取に |

---

### ジャイロ操作 vs スティック操作

#### 結論: ジャイロ操作がおすすめ

| 比較項目 | ジャイロ | スティック |
|---------|--------|-----------|
| **エイム精度** | 非常に高い | 低い |
| **反応速度** | 速い（手首の動きで即座に照準移動） | 遅い（スティック倒しに時間がかかる） |
| **慣れるまでの時間** | やや長い（1〜2週間） | 短い |
| **上位プレイヤーの使用率** | 90%以上 | 10%未満 |
| **おすすめ度** | ★★★★★ | ★★☆☆☆ |

ジャイロ操作は**コントローラーを傾けて照準を合わせる**操作方式。最初は違和感があるが、慣れると圧倒的にエイムが正確になる。上位プレイヤーのほぼ全員がジャイロを使用している。

### おすすめ感度設定

#### ジャイロ操作の場合

| 設定項目 | 初心者おすすめ | 中級者おすすめ | 上級者参考 |
|---------|-------------|-------------|-----------|
| **ジャイロ感度** | -2.5〜-1.0 | -1.0〜+2.0 | +2.0〜+5.0 |
| **Rスティック感度** | +1.0〜+3.0 | +3.0〜+5.0 | +3.0〜+5.0 |

#### 感度設定のポイント

- **ジャイロ感度は低めからスタート**: -2.5くらいから始めて、徐々に上げる
- **Rスティック感度は少し高め**: 振り向きなど大きな視点移動はスティックで行うため
- **武器種で感度を変える**: チャージャーは低め（精密射撃）、ローラーは高め（素早い振り向き）

#### 武器種別のおすすめ感度

| 武器種 | ジャイロ感度目安 | 理由 |
|--------|----------------|------|
| **シューター** | -1.0〜+2.0 | バランス型。中距離のエイム重視 |
| **チャージャー** | -3.0〜-1.0 | 低感度で精密射撃 |
| **ローラー** | +1.0〜+3.0 | 高感度で素早い振り向き＋奇襲 |
| **ブラスター** | -1.0〜+1.0 | 中感度で爆風を当てやすく |
| **スピナー** | -1.0〜+2.0 | チャージ中の微調整が重要 |
| **マニューバー** | +0.0〜+3.0 | スライド後のエイム修正に高感度が有利 |

### ジャイロ操作の練習方法

#### ステップ1: 止まってエイム練習

1. 試し打ち場（ロビーの的）で**止まった状態で的に当てる**
2. 正面→左→右→上の的に順番にエイムを合わせる
3. Y ボタンで視点リセットを活用

#### ステップ2: 移動しながらエイム

1. Lスティックで移動しながら的に当てる
2. 前後左右に動きながら照準を維持する練習
3. イカセンプクから出て即座にエイムを合わせる

#### ステップ3: 実戦練習

1. **サーモンラン**で練習（味方がいるので安心）
2. **ナワバリバトル**で実践（勝敗を気にせず練習）
3. 対面で相手を追いかけながら撃つ練習

#### ステップ4: 応用テクニック

- **イカロールエイム**: イカロール直後に素早くエイムを合わせる
- **偏差撃ち**: 動く相手の進行方向に先読みして撃つ
- **曲げ撃ち**: 撃ちながら照準を動かして横の敵も倒す
