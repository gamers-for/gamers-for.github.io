## 初心者におすすめの武器と上達のコツ

スプラトゥーン3を始めたばかりのプレイヤー向けに、**扱いやすさ**と**強さ**のバランスが取れた武器を厳選して紹介。

### スプラトゥーン3とは？

{{< youtube f22LJ66USIQ >}}

スプラトゥーン3はヒト状態とイカ状態に変化しながら戦うアクションシューティングゲーム。イカ状態ではインクの中を高速移動でき、ヒト状態ではインクを撃って塗ったり敵を倒したりできる。

### 初心者向け武器種の選び方

初心者におすすめの武器は、**シューター（銃）**か**スロッシャー（バケツ）**。シューターはシンプルな銃タイプの武器で扱いやすく、スロッシャーはバケツの中身を放り投げる広範囲攻撃ができる。

気になった武器種があれば好きな武器を使ってみるのもよい。いろんな武器を触るうちに、好みの性能の武器と出会えるはず。

**チャージャーやスピナーは上級者向け**: 銃身が長い「チャージャー」やガトリングのような「スピナー」は操作に慣れるまではおすすめできない。立ち回りが特殊で、基本操作に慣れてないうちから触っても良い動きがしづらい。

### おすすめ武器の詳細

#### わかばシューター — 最初の相棒

![わかばシューター](/images/games/splatoon3/weapons/わかばシューター.webp)

| 項目 | 内容 |
|------|------|
| **武器種** | シューター |
| **解放条件** | 最初から所持 |
| **サブ** | [スプラッシュボム](../../sub-weapons/) |
| **スペシャル** | [グレートバリア](../../special-weapons/) |

インク効率が良く、塗りの性能が高い武器。インク効率が良いのでサブのスプラッシュボムも使いやすく、短めの射程を補った攻撃も可能。真正面からの撃ち合いには向いていないので、敵が来たらサブとスペシャルを利用して攻撃を防ぎつつ、自分が戦いやすい場所に移動しよう。

**おすすめギア**: インク効率アップ(メイン)、サブ性能アップ

#### スプラシューター — 万能型の代表格

![スプラシューター](/images/games/splatoon3/weapons/スプラシューター.webp)

| 項目 | 内容 |
|------|------|
| **武器種** | シューター |
| **解放条件** | ランク2で解放 |
| **サブ** | [キューバンボム](../../sub-weapons/) |
| **スペシャル** | [ウルトラショット](../../special-weapons/) |

スプラトゥーンを代表するオールラウンダーな武器。離れすぎたり近すぎたりすると強みを活かせないため、3発でキルを取れる適正距離を理解し、仕掛けられそうなタイミングを逃さないようにしよう。「弾が届かなさすぎて負ける」「火力で押し負ける」といった問題があるときは適正距離を間違えている可能性がある。

**おすすめギア**: インク効率アップ(メイン)、サブ性能アップ

#### N-ZAP85 — 塗り効率トップクラス

![N-ZAP85](/images/games/splatoon3/weapons/n-zap85.webp)

| 項目 | 内容 |
|------|------|
| **武器種** | シューター |
| **解放条件** | ランク4で解放 |
| **サブ** | [キューバンボム](../../sub-weapons/) |
| **スペシャル** | [エナジースタンド](../../special-weapons/) |

塗り効率が非常に高く、スペシャルのエナジースタンドで味方全体を強化できるサポート性能も優秀。初心者でも塗りで貢献しやすく、チームに確実に貢献できる武器。

#### .52ガロン — 高火力シューター

![.52ガロン](/images/games/splatoon3/weapons/52ガロン.webp)

| 項目 | 内容 |
|------|------|
| **武器種** | シューター |
| **解放条件** | ランク11で解放 |
| **サブ** | [スプラッシュシールド](../../sub-weapons/) |
| **スペシャル** | [メガホンレーザー](../../special-weapons/) |

威力が高い武器だが連射力は低く射程も並。攻める際にはインクシールドを使い安全を確保しながら攻めよう。インクシールドは展開に若干時間がかかるため、展開までは相手の射程に入らないようにし、シールドに密着しながら戦う際はボムが着弾直後に爆発することを意識して、飛んできたらすぐに避けられるようにしよう。

**おすすめギア**: インク効率アップ(メイン)、イカダッシュ速度アップ

### ジャイロ操作の練習方法

{{< youtube es8Ixc0HB6A >}}

試し打ち場でジャイロ操作（コントローラーを傾けて照準を合わせる操作）の練習をしよう。まずは的にゆっくり照準を合わせるところから始める。

{{< youtube UqbMBItO9Kg >}}

ジャイロ操作に慣れてきたら、的に照準を合わせる速度を上げたり、実践に近い動きを加えて撃ってみよう。

### 武器種ごとの初心者向け解説

#### シューター — まずはここから

インクを連射する最もオーソドックスな武器種。エイムの基本を学ぶのに最適で、塗りもキルもバランス良くこなせるのでゲームの基本が身につく。

**おすすめ**: [わかばシューター](../../weapons/わかばシューター/)（最初から使える）、[スプラシューター](../../weapons/スプラシューター/)（万能型の代表格）

#### ローラー — 一撃の爽快感

転がして広範囲を塗り、振り下ろしで一撃キルを狙える武器。エイムに自信がなくても活躍しやすい。潜伏からの奇襲で勝てた時の達成感は格別。

**おすすめ**: [スプラローラー](../../weapons/スプラローラー/)（バランス型）、[カーボンローラー](../../weapons/カーボンローラー/)（軽くて扱いやすい）

#### スロッシャー — 段差に強い

バケツの中身を放り投げる広範囲攻撃ができる武器種。段差や壁越しの攻撃が得意で、エイムが苦手でも当てやすい。

**おすすめ**: [ヒッセン](../../weapons/ヒッセン/)（扱いやすい）、[バケットスロッシャー](../../weapons/バケットスロッシャー/)（攻防バランス型）

#### マニューバー — 動き回って戦いたい人に

スライドで回避しながら攻撃できる二丁拳銃スタイル。アクションゲームが好きな人に特におすすめ。

**おすすめ**: [スプラマニューバー](../../weapons/スプラマニューバー/)（入門向け）、[スパッタリー](../../weapons/スパッタリー/)（機動力特化）

### 初心者が避けるべき武器種

- **チャージャー** — エイム精度が要求される上級者向け。まずはシューターで腕を磨いてから
- **ブラスター** — 爆風の当て感覚にクセがある。シューターに慣れてからがおすすめ
- **スピナー** — チャージの立ち回りが独特。基本操作に慣れてから挑戦しよう

### 上達のコツ

1. **まず塗ることを意識** — キルよりも塗りが大事。足場を確保して有利な位置を取ろう
2. **1つの武器を使い込む** — コロコロ変えるより1本に絞って200戦ほどプレイしよう
3. **やられた原因を考える** — 相手の武器・位置・タイミングを振り返る癖をつける
4. **ギアパワーの基本を覚える** — [イカダッシュ速度アップ](../../gear-powers/)、[インク効率アップ](../../gear-powers/)、[スペシャル増加量アップ](../../gear-powers/)が汎用的
5. **試し打ちでエイム練習** — 毎日5分でも的当て練習すると上達が早い
