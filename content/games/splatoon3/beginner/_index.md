## 初心者攻略ガイド

スプラトゥーン3を始めたばかりの方に向けた総合ガイドです。

### 序盤の進め方

1. **キャラクター作成** — 見た目は後からいつでも変更可能
2. **チュートリアル完了** — 基本操作を覚える
3. **ジャイロ操作をONに** — 設定から推奨。慣れると精度が段違い
4. **試し打ちで練習** — ロビーの試し打ち場で照準と移動の練習
5. **ヒーローモード** — ストーリーモードで基礎を学ぶのがおすすめ
6. **オンライン対戦** — ナワバリバトルから参加
7. **武器選択** — 何戦かしてから自分に合う武器を決める

### 基本操作

| ボタン | 操作 |
|--------|------|
| Lスティック | 移動 |
| Rスティック | 視点移動 |
| ZRボタン | メイン武器で射撃 |
| ZLボタン | イカセンプク（インクに潜る） |
| Bボタン | ジャンプ |
| Rボタン | サブウェポン発射 |
| Rスティック押し込み | スペシャルウェポン発動 |
| Xボタン → Aボタン | スーパージャンプ（味方の位置へ飛ぶ） |

### 重要テクニック

- **イカロール** — 進行方向と逆にスティック＋Bでアーマーが付与される。対面で重要
- **イカノボリ** — 壁でBボタン長押しで素早く上昇。高台への素早いアクセスに

### おすすめ初心者武器

| 武器 | 理由 |
|------|------|
| スプラシューター | 解放ランク2。扱いやすくキルも取りやすいバランス型 |
| N-ZAP85 | 塗り性能と支援能力が高く、チームに貢献しやすい |
| わかばシューター | ランク1で使える。塗り力が高く、グレートバリアで安全に戦える |
| バレルスピナー | 長射程で後方から安全にキルが取れる |

### 上達のコツ

1. **塗りを意識する** — キルより塗りが重要。塗ることでスペシャルが溜まる
2. **マップを見る** — Xボタンでマップを確認。味方の位置と塗り状況を把握
3. **デスを減らす** — 無理に突っ込まない。生存がチームへの最大の貢献
4. **イカセンプクを活用** — インクに潜って移動・回復。常に使うのが基本
5. **試し打ちで練習** — 毎日5分の試し打ちで照準が安定する
