### ガチアサリで強い武器の特徴

1. **機動力が高い武器** — アサリがステージ全体に散らばるため、素早く回収できる移動力が重要。マニューバー系、フデ系が有利
2. **ゴール前の乱戦に強い武器** — ゴール攻めは狭い範囲での戦闘になる。ブラスター系、ローラー系の近距離火力が活きる
3. **ガチアサリを作りやすい武器** — アサリ8個でガチアサリ完成。素早く集めて突入できる機動力重視の武器が有利
4. **自陣ゴール防衛が強い武器** — 長射程でゴール前を守れるチャージャー、スピナー系。トラップやスプリンクラーで接近を検知

### ガチアサリの重要仕様

| 仕様 | 内容 |
|------|------|
| ガチアサリ完成 | アサリ8個で「ガチアサリ」に変化 |
| ゴールシュート | ガチアサリを投げ入れるとバリア破壊 + 20pt |
| バリア破壊中 | 通常アサリも投入可能（1個3pt） |
| チャンスタイム | バリアが壊れている間、追加投入で得点 |
| ガチアサリ保持 | 一定時間持つと味方・敵に位置がバレる |
