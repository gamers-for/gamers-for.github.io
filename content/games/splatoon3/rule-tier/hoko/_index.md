### ガチホコで強い武器の特徴

1. **ホコ割りが速い武器** — DPS（秒間ダメージ）が高い武器。スピナー系、シューター系がホコバリアを素早く割れる
2. **塗り性能が高くルートを作れる武器** — ホコが通る道を塗って確保する必要がある。シャープマーカー系、わかばシューターなど
3. **長射程で護衛できる武器** — ホコ持ちが進む間、遠くから敵を排除。チャージャー、スピナーが護衛に最適
4. **機動力があり先行できる武器** — マニューバー系、フデ系。ホコより先に走って道を作り、敵を排除する先行役

### ホコの重要仕様

| 仕様 | 内容 |
|------|------|
| ホコバリアHP | 約1000ダメージで割れる |
| ホコショット | ホコ持ち時にZRでチャージ射撃（威力大） |
| 自爆カウント | ホコを持って60カウントで自爆（即デス） |
| ホコ持ちの速度 | 通常より移動速度が低下 |
