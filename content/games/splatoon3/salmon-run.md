## サーモンラン（クマサン商会）攻略

![バナー](/gamers-for/images/games/splatoon3/banners/banner_000.webp)

サーモンランは4人協力でシャケの群れと戦うPvEモード。金イクラを集めてノルマを達成すれば勝利。

### 基本ルール

- **WAVE制**: 3WAVEをクリアすれば勝利
- **金イクラ**: オオモノシャケを倒すと出現。コンテナに回収
- **ノルマ**: 各WAVEで指定された数の金イクラを集める
- **制限時間**: 各WAVE 100秒
- **やられると**: 味方がインクを当てて救助可能

### オオモノシャケの倒し方

| オオモノシャケ | 弱点・倒し方 |
|---------------|-------------|
| **モグラ** | ボムを口に投げ入れる |
| **テッパン** | 背後に回り込んで弱点を攻撃 |
| **ヘビ** | 尾のパイロットを狙って攻撃 |
| **タワー** | 鍋を1段ずつ破壊する |
| **バクダン** | 爆弾を投げる瞬間に攻撃して反射 |
| **コウモリ** | 発射した弾を打ち返す |
| **カタパッド** | ミサイルランチャーにボムを投げ入れる |
| **ハシラ** | 柱に登って金網上の本体を攻撃 |
| **ダイバー** | 着地点を塗って迎撃 |
| **テッキュウ** | 大砲を使って砲撃 |
| **ナベブタ** | 円盤の下から攻撃 |
| **キンシャケ** | 光っている個体を追いかけて倒す |

### 立ち回りのコツ

#### WAVE1（序盤）
- 金イクラ回収を優先。ノルマは余裕を持ってクリアしたい
- オオモノシャケはコンテナ付近で倒すと金イクラが集まりやすい

#### WAVE2（中盤）
- 特殊WAVEに注意（ラッシュ、グリル、霧、満潮など）
- スペシャルウェポンは温存せずピンチで使う

#### WAVE3（終盤）
- 全力で生存を優先。やられたら味方に大きな負担がかかる
- タワーやカタパッドは真っ先に処理する

### ランク別アドバイス

- **かけだし〜はんにんまえ**: 金イクラ回収を優先。味方と固まって行動する
- **いちにんまえ〜じゅくれん**: オオモノシャケの処理順を意識。タワー・カタパッド最優先
- **たつじん以上**: 寄せ（コンテナ付近でオオモノを処理）を意識。無駄なデスを避ける
