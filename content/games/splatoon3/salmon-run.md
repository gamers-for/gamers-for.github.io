## サーモンラン攻略

4人協力で金イクラを集めるバイトモード。オオモノシャケを倒し、3ウェーブのノルマ達成を目指します。

---

## オオモノシャケ 優先順位と倒し方

### 最優先（Xランク）— 見つけ次第すぐ処理

| オオモノ | 倒し方 | なぜ最優先？ |
|---------|--------|------------|
| テッキュウ | 遠距離からメインで集中攻撃 | 砲弾が継続ダメージ。放置で壊滅 |
| カタパッド | ミサイル発射口にボムを投げ込む（左右2回） | ミサイルで行動が大幅に制限される |
| タワー | 鍋部分を撃って段を落とす | 追尾レーザーが極めて厄介 |

### 高優先（S〜S+ランク）

| オオモノ | 倒し方 | 特徴 |
|---------|--------|------|
| コウモリ | 吐き出した弾を撃ち返すと一撃。本体露出時を狙う | 雨弾で広範囲を塗られる |
| ヘビ | 尻尾の操縦手を攻撃 | 長い体が移動を制限する |
| ハシラ | 柱に登ってメインで攻撃 | 高所から周囲を攻撃してくる |

### 中優先（A〜Bランク）

| オオモノ | 倒し方 | 特徴 |
|---------|--------|------|
| バクダン | ボムを作る瞬間に頭部を集中攻撃 | 体には攻撃が通らない |
| テッパン | 盾にインクを当てて停止→背後から本体を攻撃 | シールド持ち |
| モグラ | 攻撃地点にボムを先置きして飲み込ませる | ウキで位置確認可能 |
| ダイバー | リング内のエリアを塗り替えて飛び込み阻止 | インクに潜む |
| ナベブタ | 落下を誘って円盤の上に乗ると楽に処理 | UFO型飛行物体 |

## 特殊ウェーブ一覧と対策

| イベント | 対策 |
|---------|------|
| ヒカリバエ（ラッシュ） | **高台に集合**。大量シャケが猛スピードで来るので高所から迎撃 |
| 霧 | 全方位警戒。オオモノが大量出現するため優先順位を意識 |
| グリル発進 | おとり役と攻撃役に分担。グリルの弱点を集中攻撃 |
| ドスコイ大量発生 | コンテナ周辺を守りながら処理 |
| キンシャケ探し | 間欠泉を順番に試す。金の光が正解 |
| ハコビヤ襲来 | 空から来る金イクラ箱を落として納品 |
| ドロシャケ噴出 | 噴出口を自インクで塗って止める |
| 巨大タツマキ | 竜巻から距離を取りつつシャケを処理 |

## オカシラシャケ

3ウェーブクリア後にランダムで出現する巨大ボス。

| オカシラ | 特徴 |
|---------|------|
| ヨコヅナ | 巨大シャケ。金イクラで投げつけてダメージ |
| タツ | 竜型。弱点を狙って攻撃 |
| ジョー | 最新追加のオカシラ |

**オカシラ連合**（3体同時出現）の倒す順番: **タツ → ヨコヅナ → ジョー**

---

## サーモンランのコツ

1. **厄介なオオモノを最優先** — テッキュウ・カタパッド・タワーは放置禁止
2. **寄せ（ルアリング）** — オオモノをコンテナ付近まで引き寄せてから倒すと納品が楽
3. **味方救助を最優先** — 倒された味方にすぐインクをかけて復活。1人ダウンで連鎖崩壊
4. **スペシャルは出し惜しみしない** — 危険な時はすぐ発動。3ウェーブで2回使える
5. **武器ごとの役割分担** — 短射程は前線でザコ処理、長射程は後方からオオモノ狙撃
