## ギア攻略まとめ

![バナー](/images/games/splatoon3/banners/banner_001.webp)

ギアは「アタマ」「フク」「クツ」の3箇所に装備でき、それぞれにギアパワーが付いている。正しいギアパワーの組み合わせが勝率に直結する。

---

### ギアの基本

| 項目 | 説明 |
|------|------|
| **装備箇所** | アタマ・フク・クツの3部位 |
| **メインギアパワー** | 各ギアに1つ。固定で変更不可（一部例外あり） |
| **サブギアパワー** | 各ギアに最大3つ。ランダムで付与 |
| **ギアパワーの効果** | メイン1つ = サブ約3.3個分の効果 |
| **ブランド** | ギアごとにブランドがあり、サブに付きやすいギアパワーが異なる |

### ギア入手方法

| 入手方法 | 説明 |
|---------|------|
| **店舗購入** | ブイヤベース（アタマ）、エビスシューズ（クツ）、エスカルゴ（フク）で日替わり商品 |
| **ゲソタウン** | Nintendo Switch Onlineアプリから注文。翌日受け取り |
| **ナワバトラー報酬** | ランクアップ報酬でギアチケット入手 |
| **サーモンラン報酬** | ランダムでギアが報酬に |
| **amiiboスキャン** | 対応amiiboで限定ギアが入手可能 |
| **フェス報酬** | フェスTはフェス限定デザイン |

### ギア厳選のやり方

1. **ブランドを選ぶ**: 欲しいサブギアパワーが付きやすいブランドを選ぶ
2. **バトルでレベルUP**: 装備して対戦するとギアの経験値が溜まる
3. **サブが3つ埋まる**: サブギアパワーが3つ付与される
4. **クリーニング**: ギアのサブをリセット（スーパーサザエで付け替え可能）
5. **スーパーサザエで付け替え**: 好きなギアパワーに変更

### ブランドとギアパワーの関係

| ブランド | 付きやすい | 付きにくい |
|---------|-----------|-----------|
| **バトロイカ** | インク効率アップ(サブ) | インク回復量アップ |
| **エゾッコ** | スペシャル減少量ダウン | スペシャル増加量アップ |
| **クラーゲス** | ヒト移動速度アップ | イカダッシュ速度アップ |
| **シグレニ** | 復活時間短縮 | スーパージャンプ時間短縮 |
| **アイロニック** | アクション強化 | サブ性能アップ |
| **ヤコ** | イカダッシュ速度アップ | ヒト移動速度アップ |
| **ロッケンベルグ** | イカダッシュ速度アップ | 相手インク影響軽減 |
| **アロメ** | インク回復量アップ | インク効率アップ(サブ) |
| **ジモン** | インク効率アップ(メイン) | スペシャル減少量ダウン |
| **ホタックス** | 相手インク影響軽減 | アクション強化 |
| **タタキケンサキ** | スーパージャンプ時間短縮 | 復活時間短縮 |
