## 定期イベント一覧

| イベント | 開催頻度 | 概要 |
|---------|---------|------|
| **フェス** | 約1〜2ヶ月に1回 | 3チームに分かれて対戦。トリカラバトル含む |
| **ビッグラン** | 不定期（年2〜3回） | 通常バトルステージにシャケが襲来する特殊サーモンラン |
| **バイトチームコンテスト** | 月1〜2回 | サーモンランのスコアアタック。ランキング形式 |

## フェス

### フェスの流れ

| フェーズ | 期間 | 内容 |
|---------|------|------|
| **お題発表** | 開催2週間前 | 3つの陣営が発表される |
| **投票期間** | 開催1週間前 | 広場で陣営を選択し、フェスTを受け取る |
| **前半戦** | 開催〜24時間 | 通常の4対4バトル（同陣営内マッチングもあり） |
| **中間発表** | 24時間経過時 | 得票率・バトル勝率の中間結果発表。1位チームが「防衛側」に |
| **後半戦** | 24時間〜終了 | **トリカラバトル**が解放。防衛側は2対4対2の三つ巴 |
| **結果発表** | 終了後 | 各部門の勝敗から総合優勝チームを決定 |

### フェスの報酬

| ランク | 必要フェスポイント | 報酬 |
|--------|------------------|------|
| えいえんのフェスパワーまで | 0p | フェスT（投票時） |
| かけだし | 0p | スーパーサザエ ×1 |
| まあまあ | 100p | スーパーサザエ ×2 |
| まことの | 350p | スーパーサザエ ×3 |
| スーパー | 600p | スーパーサザエ ×5 |
| カリスマ | 999p | スーパーサザエ ×10 |

## ビッグラン

通常のバトルステージにシャケが襲来する特殊イベント。通常のサーモンランとは異なるステージで行われるため、新鮮な体験ができる。

### ビッグランの特徴

| 項目 | 内容 |
|------|------|
| **ステージ** | 通常バトルステージ（ユノハナ大渓谷、マテガイ放水路 等） |
| **オオモノシャケ** | 通常サーモンランと同じオオモノが出現 |
| **オカシラシャケ** | ビッグラン限定で出現。討伐するとウロコ大量獲得 |
| **報酬** | 累計イクラ数でランキング。上位入賞でステッカーやプレート |
| **開催期間** | 約48時間（土日開催が多い） |

### 過去のビッグラン開催履歴

| 回数 | 開催時期 | ステージ |
|------|---------|---------|
| **第1回** | 2022年12月 | ユノハナ大渓谷 |
| **第2回** | 2023年4月 | マテガイ放水路 |
| **第3回** | 2023年9月 | タラポートショッピングパーク |
| **第4回** | 2023年12月 | チョウザメ造船 |
| **第5回** | 2024年4月 | スメーシーワールド |
| **第6回** | 2024年8月 | 海女美術大学 |

## バイトチームコンテスト

4人チームでサーモンランのハイスコアを競うランキングイベント。

### バイトチームコンテストの仕組み

| 項目 | 内容 |
|------|------|
| **参加方法** | サーモンランの「チームコンテスト」を選択 |
| **マッチング** | フレンドまたは野良4人でチーム結成 |
| **ルール** | 3WAVE中のイクラ納品数の合計で競う |
| **ランキング** | 全参加チーム中の順位で報酬変動 |
| **開催期間** | 48時間（主に土日） |

### ランキング報酬

| 順位帯 | 報酬 |
|--------|------|
| 上位5% | 金プレート + ステッカー |
| 上位20% | 銀プレート + ステッカー |
| 上位50% | 銅プレート |
| 全参加者 | 参加ステッカー |

## 季節限定コンテンツ

### シーズンで変わるもの

| コンテンツ | 説明 |
|-----------|------|
| **カタログ** | シーズンごとにカタログがリセット。レベルUPで報酬獲得 |
| **ロッカーアイテム** | シーズン限定のロッカー装飾アイテムが追加 |
| **フェス T シャツ** | フェスごとに限定デザイン |
| **ナワバトラーカード** | 新武器追加に伴い新カード追加 |
| **ゲソタウン** | 月替わりで限定ギアが登場 |

### カタログの報酬

カタログはシーズンごとにレベル1からスタート。

| レベル帯 | 主な報酬 |
|---------|---------|
| Lv.1〜10 | エモート、ステッカー |
| Lv.11〜30 | ギアの引換券、ロッカーアイテム |
| Lv.31〜50 | 限定ギア、カードパック |
| Lv.51〜100 | レアギア、称号パーツ |
| Lv.100以降 | おカネ（ゲーム内通貨）のみ |
