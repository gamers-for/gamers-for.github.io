## ナワバトラー攻略

![バナー](/images/games/splatoon3/banners/banner_001.webp)

ナワバトラーはスプラトゥーン3に搭載されたカードゲーム。1対1で行う陣取りゲームで、武器をモチーフにした「ナワバトラーカード」を使ってマスを奪い合う。

---

### ナワバトラーの基本ルール

| 項目 | 内容 |
|------|------|
| **プレイ人数** | 1対1 |
| **ターン数** | 12ターン |
| **勝利条件** | 最終的に自分の色のマスが多い方が勝ち |
| **カードの使い方** | 手札から1枚選び、盤面に配置 |
| **スペシャルポイント（SP）** | カードの角が相手の色と隣接するとSPが溜まる |
| **スペシャル発動** | SPが溜まるとスペシャルアタックで相手のマスを塗り替え可能 |
| **パス** | カードを出さずにパスも可能（SP+1） |

### カードの見方

各カードには以下の情報がある。

| 要素 | 説明 |
|------|------|
| **マス形状** | カードごとに固有の形。大きいほど一度に多くのマスを塗れる |
| **SPコスト** | スペシャルアタックに必要なSPの量 |
| **レアリティ** | コモン（灰）→レア（緑）→フレッシュ（紫）→ローヤル（金）の4段階 |
| **マス数** | 少ないもので3マス、多いもので15マス以上 |

---

### デッキ構成のコツ

デッキは**15枚**で構成する。以下のバランスが重要。

#### おすすめデッキ構成比率

| カード種別 | 枚数目安 | 役割 |
|-----------|---------|------|
| **大型カード（10マス以上）** | 3〜4枚 | 序盤の大量塗り、陣地確保 |
| **中型カード（6〜9マス）** | 5〜6枚 | メインの塗り。柔軟に配置 |
| **小型カード（3〜5マス）** | 3〜4枚 | 隙間埋め、SP稼ぎ |
| **SP用カード（コスト低め）** | 2〜3枚 | スペシャルアタック用 |

#### デッキ構築のポイント

- **大型カードを入れすぎない**: 手札事故（置ける場所がない）が起きやすくなる
- **小型カードも重要**: 終盤の隙間埋めや角の接触でSP稼ぎに不可欠
- **SPコストのバランス**: コスト3以下のカードを5枚以上入れると安定
- **L字・T字型のカードが強い**: 角や壁際にフィットしやすい

### 最強デッキレシピ

#### 安定型バランスデッキ

序盤に大型で陣地を確保し、中盤以降は中・小型で隙間を埋める基本デッキ。

| 枠 | カード例 | マス数 | 役割 |
|----|---------|--------|------|
| 大型1 | ハイドラント系 | 13マス | 序盤の大量塗り |
| 大型2 | ダイナモローラー系 | 12マス | 中央制圧 |
| 大型3 | リッター系 | 11マス | 縦長で壁際攻め |
| 中型1〜5 | シューター系各種 | 6〜9マス | 主力の塗り |
| 小型1〜4 | スプラッシュボム系 | 3〜5マス | 隙間埋め・SP稼ぎ |
| SP用1〜3 | SPコスト3以下 | 任意 | スペシャル発動 |

#### SP特化デッキ

スペシャルアタックを多用して相手のマスを塗り替える攻撃的デッキ。

- 小型カード多め（8〜9枚）でSPを高速で溜める
- スペシャルアタックで相手の大型カード塗りを無効化
- 相手の角に隣接するよう意識して配置

---

### 勝つためのテクニック

#### 序盤（1〜4ターン）

- 大型カードを先に出して自陣側を確保
- マップ中央ではなく端から攻めると安定
- 角に自分のカードを多く接触させてSP確保

#### 中盤（5〜8ターン）

- 相手の進行方向をブロック
- 中型カードで陣地を広げる
- SPが溜まったらスペシャルアタックのタイミングを計る

#### 終盤（9〜12ターン）

- 小型カードで隙間を埋める
- 残りターン数とSP量を計算して最適なタイミングでスペシャル発動
- パスを戦略的に使う（置ける場所がないなら無理に置かない）

#### スペシャルアタックの使い方

- **発動タイミング**: 相手の大きな塗りを塗り替えられる時
- **角12マス**: スペシャルは隣接する角を起点に12マスを自分の色に変える
- **先に使わない**: 相手にスペシャルで返される可能性があるので、後出しが基本

### ランクシステム

ナワバトラーにはランクがあり、NPC対戦で上がる。

| ランク | 解放条件 | 報酬 |
|--------|---------|------|
| ランク1〜10 | NPC対戦で勝利 | カードパック |
| ランク11〜20 | 上位NPCに勝利 | 限定カードパック、称号 |
| ランク21〜30 | さらに上位NPCに勝利 | レアカードパック |
| ランク30以上 | 全NPCに勝利後、通信対戦で上昇 | 限定称号、ロッカーアイテム |

### カードの入手方法

| 入手方法 | 説明 |
|---------|------|
| **カードパック** | ナワバトラーの勝利報酬やショップで購入 |
| **カードのかけら** | NPCに勝利時や報酬で入手。一定数集めるとカードに変換 |
| **amiiboスキャン** | 対応amiiboでカード入手 |
| **ロブの店** | ロブの店の景品としてカードパックが並ぶことがある |
| **フェス報酬** | フェスのスーパーサザエ交換でカードパック入手可能 |

### 通信対戦

- ロビーのナワバトラー台でフレンドやランダムマッチング対戦が可能
- 通信対戦ではランクポイントを獲得
- 相手のデッキを見てからのデッキ変更は不可
