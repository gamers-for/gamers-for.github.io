## 最強スペシャルウェポンランキング

![バナー](/images/games/splatoon3/banners/banner_001.webp)

全スペシャルウェポンを、ルール汎用性・味方貢献度・単体での制圧力を基準にランク付け。

### 評価基準

| 基準 | 説明 |
|------|------|
| **ルール汎用性** | ナワバリ・バンカラ全ルールで活躍できるか |
| **味方貢献度** | チーム全体の打開・防衛に貢献できるか |
| **単体の強さ** | 1人で使っても十分な効果を発揮するか |

---

### SSランク — 環境トップのスペシャル

| スペシャル | 評価理由 |
|-----------|---------|
| **[ウルトラショット](../special-weapons/#ウルトラショット)** | 超長射程の3発弾。安全な位置から打開でき、遠距離のチャージャーも倒せる。全ルールで最強クラス |
| **[トリプルトルネード](../special-weapons/#トリプルトルネード)** | 3箇所に竜巻を設置。エリア塗り・ゾーン制圧・打開全てに使える万能スペシャル |
| **[エナジースタンド](../special-weapons/#エナジースタンド)** | 味方全員にスピードアップ＋インク回復バフ。打開時に使うとチーム全体の戦力が大幅UP |

### S+ランク — 非常に強力

| スペシャル | 評価理由 |
|-----------|---------|
| **[カニタンク](../special-weapons/#カニタンク)** | 高耐久＋連射＋カノン砲。前線制圧力が最強。打開の要 |
| **[ナイスダマ](../special-weapons/#ナイスダマ)** | アーマー付き＋広範囲爆破。味方のナイス連打で威力UP。防御と攻撃を両立 |
| **[マルチミサイル](../special-weapons/#マルチミサイル)** | 安全圏から複数の敵をロックオン。敵の位置把握＋行動制限が強い |
| **[ジェットパック](../special-weapons/#ジェットパック)** | 空中から爆撃。高所制圧＋ヤグラ防衛に特に強い |

### Sランク — 安定して強い

| スペシャル | 評価理由 |
|-----------|---------|
| **[メガホンレーザー5.1ch](../special-weapons/#メガホンレーザー51ch)** | 追尾レーザーで広範囲制圧。エリア確保・防衛で安定した効果 |
| **[グレートバリア](../special-weapons/#グレートバリア)** | 巨大シールド展開。ヤグラ・ホコで防衛の要。エリア確保にも |
| **[テイオウイカ](../special-weapons/#テイオウイカ)** | 巨大イカで突進。無敵＋高速移動。逃走や奇襲に |

### Aランク — ルール次第で活躍

| スペシャル | 評価理由 |
|-----------|---------|
| **[アメフラシ](../special-weapons/#アメフラシ)** | 雨雲でじわじわダメージ。エリア確保の補助に。打開力はやや低い |
| **[ホップソナー](../special-weapons/#ホップソナー)** | 索敵＋衝撃波。センサー効果で味方全員に敵位置を共有 |
| **[デコイチラシ](../special-weapons/#デコイチラシ)** | デコイで敵を撹乱。打開時の陽動や味方の進軍サポートに |
| **[ウルトラチャクチ](../special-weapons/#ウルトラチャクチ)** | 広範囲のチャクチ。着地点の塗りが強力。対面拒否に使える |

### Bランク — 使いこなすと強い

| スペシャル | 評価理由 |
|-----------|---------|
| **[キューインキ](../special-weapons/#キューインキ)** | 敵のインクを吸収して撃ち返す。対スペシャル性能が高いが、使いどころが難しい |
| **[ウルトラハンコ](../special-weapons/#ウルトラハンコ)** | 前方に連続叩きつけ＋投げ。通路制圧向き。横からの攻撃に弱い |
| **[スミナガシート](../special-weapons/#スミナガシート)** | 壁を設置して視界遮断。エリアの分断や奇襲の準備に |

### Cランク — 環境的に不利

| スペシャル | 評価理由 |
|-----------|---------|
| **[サメライド](../special-weapons/#サメライド)** | サメで突進して爆発。突進中に倒されることが多く、安定しない |
| **[ショクワンダー](../special-weapons/#ショクワンダー)** | ワイヤーで飛んで奇襲。使いこなせれば強いが、戻り先が固定で狩られやすい |

---

### スペシャルの使い方の基本

#### 打開時（攻め）

1. 味方と**スペシャルのタイミングを合わせる**（2〜3人同時が理想）
2. ウルトラショットやトリプルトルネードで**敵を退かせてから前進**
3. カニタンクやテイオウイカで**前線を押し上げる**

#### 防衛時（守り）

1. グレートバリアで**エリアやヤグラを守る**
2. マルチミサイルで**敵の進軍を遅らせる**
3. ナイスダマのアーマーで**生存しながら迎撃**

#### カウンター（切り返し）

1. 敵のスペシャルに対してキューインキで**吸収して返す**
2. ホップソナーで**潜伏中の敵を暴く**
3. メガホンレーザーで**複数の敵をまとめて動かす**
