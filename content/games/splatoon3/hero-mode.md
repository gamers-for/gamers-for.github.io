## ヒーローモード攻略

![バナー](/images/games/splatoon3/banners/banner_004.webp)

ヒーローモードはスプラトゥーン3の1人用ストーリーモード「Return of the Mammalians」。オルタナと呼ばれる地下世界を冒険し、毛むくじゃらのタコたちの謎に迫る。オンライン対戦の練習にもなるため、初心者は最初にプレイするのがおすすめ。

### 基本情報

| 項目 | 内容 |
|------|------|
| **プレイ人数** | 1人 |
| **舞台** | オルタナ（地下世界） |
| **主人公** | 3号（プレイヤー） |
| **相棒** | コジャケ（小さなサケ） |
| **難易度** | 普通（一部ステージは高難度） |
| **クリア報酬** | 限定ギア、称号、ロッカーアイテム、ブキチドローン |

### ヒーローモードへの行き方

広場のマンホールからオルタナに行くことができる。チュートリアル完了後、いつでもアクセス可能。マンホールはロビーから広場に出て右手に進んだ場所にある。

### コジャケ（相棒）の活用

コジャケはプレイヤーをサポートするパートナー。毛の色を変えることで特殊能力が変化する。

| 毛の状態 | 能力 |
|---------|------|
| **ケバケバ** | 敵に向かって突進攻撃をする |
| **ショート** | インクで周囲を塗ってくれる |
| **ナガナガ** | インクを振り回して広範囲を攻撃 |

コジャケにイクラを与えることで能力を強化できる。ステージに応じて毛の状態を使い分けるのがコツ。

### 全エリア一覧

#### サイト1: 未来のユートピアランド

最初のエリア。基本操作を学べるチュートリアル的なステージが中心。

| ステージ | 内容 |
|---------|------|
| **ステージ1〜4** | 基本的な移動・射撃・イカ状態の使い方を学ぶ |
| **ステージ5〜7** | サブウェポンやスペシャルを使ったステージ |
| **ボス** | タコツボザムライ |

#### サイト2: ロケット紡績工場

難易度が少し上がるエリア。壁塗りやレールなどのギミックが増える。

| ステージ | 内容 |
|---------|------|
| **ステージ8〜14** | レール移動、壁塗りジャンプなど応用テクニックを活用 |
| **ボス** | オクトシャワー |

#### サイト3: 永遠のモグモグハウス

イカロールなどの新アクションを活用するステージが登場。

| ステージ | 内容 |
|---------|------|
| **ステージ15〜21** | イカロール・イカノボリを使うステージが多い |
| **ボス** | オクトスタンプ |

#### サイト4: 未来パフォーマンスセンター

難易度がさらに上昇。複雑なギミックと強い敵が登場。

| ステージ | 内容 |
|---------|------|
| **ステージ22〜28** | タコゾネスとの戦闘や複合ギミックステージ |
| **ボス** | オクトスナイパー |

#### サイト5: あすなろ宇宙基地

高難易度ステージが連続するエリア。

| ステージ | 内容 |
|---------|------|
| **ステージ29〜35** | 制限時間つきステージやボスラッシュ的な構成 |
| **ボス** | オクトバース |

#### サイト6: オルタナ最深部

最終エリア。ストーリーのクライマックス。

| ステージ | 内容 |
|---------|------|
| **最終ステージ** | ラスボス「DJタコワサ将軍」との決戦 |

### ボス攻略

| ボス | 弱点・攻略法 |
|------|------------|
| **タコツボザムライ** | 背中の弱点にインクを当てる。攻撃パターンは単純なので回避しやすい |
| **オクトシャワー** | 上部のシャワーヘッドを撃ち落とす。足場が動くので位置取りに注意 |
| **オクトスタンプ** | 突進をかわして背面の弱点を攻撃。突進後の隙が大きい |
| **オクトスナイパー** | 遮蔽物を利用して接近し弱点を攻撃。照準に当たらないよう移動し続ける |
| **オクトバース** | 各形態の攻撃パターンを覚えて回避しつつ弱点を攻撃 |
| **DJタコワサ将軍** | 複数フェーズの戦闘。各フェーズで攻撃パターンが変化する最難関ボス |

### イクラ稼ぎのコツ

- 各ステージ内に隠されたイクラを探す（壁の裏や高い場所にあることが多い）
- ステージのやり直しで再度イクラを回収可能
- ケバインクで覆われたオブジェクトをコジャケで浄化するとイクラが出現
- 効率的にイクラを集めてコジャケやブキをアップグレードしよう

### 報酬一覧

| 報酬 | 入手条件 |
|------|---------|
| **限定ギア** | ストーリークリアで入手 |
| **称号** | 各エリアのクリアやチャレンジ達成で解放 |
| **ロッカーアイテム** | ステージ内の隠しアイテムを発見 |
| **ブキチドローン** | 全ステージクリア後に解放。オンラインで使用可能 |

### 全クリア後の要素

- すべてのステージをクリアすると特別なエンディングが見られる
- **全ブキクリア**: 各ステージを全種類の武器でクリアするやりこみ要素
- 全ブキクリア報酬は存在するかは公式未発表。達成感が最大の報酬
- 隠しステージの発見やイクラ100%回収なども挑戦要素

### 初心者への推奨プレイ順

1. **まずヒーローモードをプレイ** — 基本操作を学べる
2. **サイト1〜3をクリア** — イカ操作・サブ・スペシャルの基本が身につく
3. **オンラインに挑戦** — ヒーローモードで学んだ技術を実戦で使う
4. **残りのサイトをクリア** — オンライン経験を活かして高難度ステージに挑む
