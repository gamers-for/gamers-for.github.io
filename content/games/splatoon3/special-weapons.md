## スペシャルウェポン一覧

スプラトゥーン3の全スペシャルウェポンの効果と使い方を解説します。

### スプラトゥーン3からの新スペシャル

| スペシャル | 効果 |
|-----------|------|
| ウルトラショット | 回転しながら飛ぶ強力ショットを最大3回発射。遠距離キル可能 |
| グレートバリア | 大型バリア装置を設置。味方を守りながら前線維持 |
| ショクワンダー | 触腕をのばしてステージを飛び回る変身。機動力が大幅UP |
| ホップソナー | ウェーブで隠れた相手をあぶりだす装置を設置 |
| キューインキ | 相手のインクを吸い込むほど強力になるインク弾を放つ |
| メガホンレーザー5.1ch | 6つのメガホン型レーザーで広範囲攻撃 |
| カニタンク | 多足歩行戦車に搭乗。ショットとカノン砲で制圧 |
| サメライド | インクのフロートで前方に突撃。着地点で爆発 |
| トリプルトルネード | 最大3か所にインクの竜巻を発生 |
| エナジースタンド | 能力強化ドリンクの冷蔵庫を設置。味方全員が恩恵 |
| テイオウイカ | 巨大なイカに変身。無敵状態で突進 |
| デコイチラシ | 大量のデコイを張り付けて陽動 |
| スミナガシート | ゆっくり前進する巨大シートを展開 |
| ウルトラチャクチ | 浮き上がって3つの爆発を発生。打開に有効 |

### スプラトゥーン2からの続投スペシャル

| スペシャル | 効果 |
|-----------|------|
| マルチミサイル | カーソル内の相手全員にミサイル発射 |
| アメフラシ | インクの雨雲を発生させる装置を投げる |
| ナイスダマ | インクの塊を作り投げつけて大爆発。使用中はアーマー付与 |
| ジェットパック | 空を飛びながらランチャーで攻撃 |
| ウルトラハンコ | 巨大なハンコで前方を叩きながら突進 |

### スペシャルの使い方のコツ

- **打開時に味方と合わせる** — 1人で使うより2〜3人で同時に使った方が効果的
- **出し惜しみしない** — 溜まったら積極的に使う。デスするとゲージが減る
- **有利時は温存も選択肢** — 相手の打開を潰すために取っておく場合もある
