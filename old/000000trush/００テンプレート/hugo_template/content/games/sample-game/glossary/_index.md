## 📖 用語集

### 基本用語

| 用語 | 読み | 意味 |
|------|------|------|
| **DPS** | ディーピーエス | Damage Per Second。1秒あたりのダメージ |
| **TTK** | ティーティーケー | Time To Kill。敵を倒すまでの時間 |
| **AoE** | エーオーイー | Area of Effect。範囲攻撃 |
| **DoT** | ディーオーティー | Damage over Time。継続ダメージ |
| **CC** | シーシー | Crowd Control。行動妨害（スタン・凍結等） |
| **CD** | シーディー | Cooldown。スキルの再使用時間 |
| **NPC** | エヌピーシー | Non Player Character。操作できないキャラ |
| **PvP** | ピーブイピー | Player vs Player。対人戦 |
| **PvE** | ピーブイイー | Player vs Environment。対CPU戦 |
| **OP** | オーピー | Over Powered。強すぎる |
| **Nerf** | ナーフ | 下方修正 |
| **Buff** | バフ | 上方修正 or 強化効果 |
| **Debuff** | デバフ | 弱体化効果 |
| **Meta** | メタ | 現環境で最も強い戦法/構成 |
| **Tier** | ティア | 強さのランク（Tier1が最強） |

### ソシャゲ用語

| 用語 | 意味 |
|------|------|
| **リセマラ** | リセットマラソン。最初のガチャを繰り返す |
| **天井** | 一定回数でピックアップ確定 |
| **すり抜け** | ピックアップ以外の☆5が出ること |
| **完凸** | 同じキャラを最大回数引いて最大強化 |
| **人権** | 持っていないと攻略困難なキャラ |
| **石** | 課金通貨（ジェム・ダイヤ等の総称） |

### FPS用語

| 用語 | 意味 |
|------|------|
| **ADS** | Aim Down Sight。覗き込みエイム |
| **ヘッショ** | ヘッドショット |
| **リコイル** | 銃の反動 |
| **ピーク** | 角から顔を出して索敵 |
| **キャンプ** | 一箇所に留まる戦法 |
| **芋** | キャンプの蔑称 |

### MOBA用語

| 用語 | 意味 |
|------|------|
| **CS** | Creep Score。ミニオンのラストヒット数 |
| **ガンク** | 他レーンへの奇襲 |
| **ロスト** | タワーを折られること |
| **ファーム** | ミニオン/中立を倒して稼ぐ |
| **フィード** | 何度も倒されて相手を育てること |

### 格闘ゲーム用語

| 用語 | 意味 |
|------|------|
| **F（フレーム）** | 1/60秒。技の速さの単位 |
| **確反** | 確定反撃。ガード後に確定で当たる技 |
| **暴れ** | 相手の攻めの隙間に技を出すこと |
| **起き攻め** | ダウンから起き上がる相手を攻めること |
| **詐欺飛び** | 対空無敵技をガードできるジャンプ攻撃 |
