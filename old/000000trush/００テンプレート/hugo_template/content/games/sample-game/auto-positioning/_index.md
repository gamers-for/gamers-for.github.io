## 📐 配置の基本

### 盤面レイアウト
```
[後衛3列目] 🔮  🔮  🔮  🏹
[中衛2列目] ___  ___  ___  ___
[前衛1列目] 🛡️  ⚔️  ⚔️  🛡️
─────────── 相手側 ───────────
```

### ポジション別適性

| 位置 | 適性タイプ | 理由 |
|------|----------|------|
| 前列・端 | 🛡️タンク | 敵の攻撃を受ける |
| 前列・中央 | ⚔️ファイター | 複数の敵と交戦 |
| 後列・端 | 🏹レンジャー | 安全に攻撃 |
| 後列・中央 | 🔮メイジ | 範囲攻撃を最大化 |
| 最後列・角 | 💀アサシン(キャリー) | 最も安全な位置 |

## 🔄 配置の調整

| 相手の構成 | 対策配置 |
|-----------|---------|
| 💀アサシン多め | キャリーを角→中央に移動 |
| 🔮メイジ多め | ユニットを散開させる |
| 🏹レンジャー多め | タンクを前に出して時間稼ぎ |
| ⚔️ファイター多め | 後衛を下げて距離を取る |
