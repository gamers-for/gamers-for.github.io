## 📐 基本配置パターン

### パターン1: 一本道型
```
敵→→→ ⚔️🛡️ →→→ 🏹🔮 →→→ ⚔️💖 →→→ 💥 →ゴール
        (前衛)    (中衛)    (後衛)   (最終防衛)
```

### パターン2: 挟み撃ち型
```
      🏹🔮💥
敵→→→→→→→→→→→→→ゴール
      🏹🔮💥
   (上下から十字砲火)
```

### パターン3: 迷路型（上級）
```
敵→┐  ┌→┐  ┌→ゴール
   │🔮│  │💥│
   └→┘  └→┘
(壁でルートを伸ばす)
```

## 💡 配置のコツ

| | コツ |
|---|------|
| 🛡️ | ブロッカーは敵ルートの曲がり角に置く |
| 🏹 | 遠距離は射程が最大限活きる位置に |
| 🔮 | 範囲攻撃は敵が密集する地点に |
| 💖 | ヒーラーはブロッカーの隣に |
