## 👥 マルチプレイモード

| モード | 人数 | 内容 |
|--------|------|------|
| 🤝Co-op（協力） | 2-4人 | ストーリーを協力プレイ |
| ⚔️PvP（対戦） | 1vs1〜5vs5 | 対人戦 |
| 👥レイド | 4-8人 | 高難度ボス討伐 |
| 🏰ギルド戦 | ギルドvsギルド | 大規模対人戦 |

## 🤝 Co-op攻略のコツ

| | コツ |
|---|------|
| 📢 | ボイスチャット or 定型文を活用 |
| 🎭 | 役割分担を決める（タンク・ヒーラー・DPS） |
| ⏰ | ピンチ時はピンで位置を知らせる |
| 💀 | 蘇生優先。味方が倒れたらすぐ助ける |

## 🔗 マッチング

| 方法 | 特徴 |
|------|------|
| 🔍オートマッチ | ランダムマッチング |
| 🔑ルームID | フレンドと合流 |
| 👥フレンド招待 | フレンドリストから直接 |
