## ⏰ 放置の仕組み

| 項目 | 内容 |
|------|------|
| 放置上限 | 最大12時間分（VIPで24時間） |
| 獲得物 | 💰ゴールド、📖経験値、⚙️素材 |
| 計算基準 | 到達した最高ステージの難易度 |

## 📊 放置効率を上げる方法

| 優先度 | 方法 | 効果 |
|--------|------|------|
| 🔴最優先 | ステージを1つでも先に進める | 基礎報酬UP |
| 🔴最優先 | 放置時間を溢れさせない | 12時間ごとに回収 |
| 🟡推奨 | VIP購入(上限24h) | 回収頻度半減 |
| 🟡推奨 | 放置報酬バフキャラ配置 | +20%〜50% |
| 🟢任意 | 2倍速チケット使用 | 一時的に2倍 |

## 🔄 転生（プレステージ）

| 項目 | 内容 |
|------|------|
| 解放条件 | ステージ100到達 |
| リセット | ステージ・レベル・装備 |
| 引き継ぎ | 💎ジェム、🌟転生ポイント、アンロック |
| メリット | 転生ポイントで永久バフ購入 |
