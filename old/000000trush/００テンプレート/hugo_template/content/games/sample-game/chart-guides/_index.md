## 📖 譜面攻略

### 💜 Violet Storm [EXPERT ⭐15]

**最高難度曲の攻略**

| セクション | 小節 | 難所 | 対策 |
|-----------|------|------|------|
| イントロ | 1-8 | 高速トリル(16分) | 脱力。力まない |
| Aメロ | 9-24 | 混フレ(3拍子→4拍子) | 曲を暗記 |
| サビ | 25-40 | 乱打+同時押し | 運指を固定 |
| ラスサビ | 41-56 | 全属性複合 | 体力温存が鍵 |
| アウトロ | 57-64 | 減速トリル | 目押しで対応 |

**クリア目安**: 地力S以上 / **フルコン目安**: 地力S+

### 🔥 Inferno [EXPERT ⭐13]

| セクション | 難所 | 対策 |
|-----------|------|------|
| イントロ | 8分乱打 | ウォーミングアップ |
| Bメロ | 16分階段 | 薬指まで使う |
| サビ | 同時押し+フリック | フリック方向注意 |

## 💡 フルコンのコツ

| | コツ |
|---|------|
| 🎧 | イヤホン必須。SE音量を大きめに |
| 👆 | 判定タイミングは曲ごとに微調整 |
| 💪 | 難所前で力を抜く。脱力が最重要 |
| 🔄 | 苦手部分だけ繰り返し練習 |
| 📱 | 端末の処理落ちに注意（重い端末は不利） |
