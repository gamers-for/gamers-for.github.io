## 🐛 既知のバグ

### 現在発生中

| 状態 | バグ内容 | 発生条件 | 対処法 |
|------|---------|---------|--------|
| 🔴未修正 | 特定ステージでフリーズ | 5章ボス戦でスキル3使用 | スキル3を使わない |
| 🟡修正予定 | アイテムが消える | インベントリ満杯時に報酬受取 | 空きを作ってから受取 |
| 🟢修正済み | ログインできない | v2.4.1以前 | アプリ更新 |

### エラーコード

| コード | 原因 | 対処法 |
|--------|------|--------|
| **E-001** | サーバー接続エラー | Wi-Fi確認→再起動→時間を空けて再試行 |
| **E-002** | メンテナンス中 | 公式Xで終了時間を確認 |
| **E-003** | バージョン不一致 | アプリストアで更新 |
| **E-004** | データ破損 | キャッシュクリア→再DL |
| **E-005** | 認証エラー | アカウント再ログイン |

## 🔧 よくあるトラブル

| トラブル | 対処法 |
|---------|--------|
| 📱アプリが起動しない | キャッシュクリア→再インストール |
| 🐌動作が重い | グラフィック設定を「低」に変更 |
| 🔇音が出ない | 端末のサイレントモード確認 |
| 📶通信エラーが頻発 | Wi-Fiを切って4G/5Gに切替 |
| 🔋バッテリー消費が激しい | フレームレートを30fpsに制限 |
