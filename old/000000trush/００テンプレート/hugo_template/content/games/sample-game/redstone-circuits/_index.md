## ⚡ レッドストーンの基本

| 部品 | 機能 | 用途 |
|------|------|------|
| 🔴レッドストーンダスト | 信号伝達 | 配線 |
| 🔦レッドストーントーチ | 信号源+NOT回路 | 論理回路 |
| 🔁リピーター | 信号延長+遅延 | タイミング制御 |
| 🔀コンパレーター | 信号比較+減算 | 高度な回路 |
| 🔘ボタン/レバー | 入力 | スイッチ |
| 🚪ドア/ピストン | 出力 | 物理的動作 |

### 基本回路

| 回路名 | 機能 | 組み方 |
|--------|------|--------|
| AND回路 | 両方ONで出力 | トーチ2つ+ダスト |
| OR回路 | どちらかONで出力 | ダスト合流 |
| NOT回路 | 反転 | トーチ1つ |
| クロック | 定期的にON/OFF | リピーター4つでループ |
