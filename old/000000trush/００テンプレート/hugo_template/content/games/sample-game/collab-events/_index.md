## 🤝 コラボイベントの基本

| 項目 | 内容 |
|------|------|
| 📅期間 | 通常2〜3週間（**復刻なしが多い**） |
| 🎁報酬 | 限定キャラ/スキン/装備（**期間限定**） |
| ⚠️注意 | **取り逃すと二度と入手できない場合あり** |
