## 📱 データ引き継ぎ方法

### 方法一覧

| 方法 | 対応 | 難易度 | おすすめ |
|------|------|--------|---------|
| 🔗アカウント連携 | iOS⇔Android⇔PC | ⭐ | ✅最もおすすめ |
| 📝引き継ぎコード | 同OS間 | ⭐⭐ | ⚠️有効期限あり |
| ☁️クラウドセーブ | 同OS間 | ⭐ | ✅自動バックアップ |

### 🔗 アカウント連携の手順

| ステップ | 旧端末 | 新端末 |
|---------|--------|--------|
| 1 | ⚙️設定→アカウント連携 | - |
| 2 | Google/Apple/X/Facebookを選択 | - |
| 3 | 連携完了を確認（**スクショ推奨**） | - |
| 4 | - | アプリインストール |
| 5 | - | タイトル画面→「データ引き継ぎ」 |
| 6 | - | 同じアカウントでログイン |
| 7 | - | ✅引き継ぎ完了 |

### ⚠️ 注意点

| | 注意 |
|---|------|
| 💎 | **有償ジェム**はOS間で引き継げない場合あり |
| 📱 | 旧端末のデータは引き継ぎ後**削除される** |
| 🔗 | 連携前に**プレイヤーIDをメモ**しておく |
| 📞 | トラブル時は**運営に問い合わせ**（プレイヤーIDが必要） |
| 📸 | 引き継ぎ前に**スクショを複数枚**撮っておく |
