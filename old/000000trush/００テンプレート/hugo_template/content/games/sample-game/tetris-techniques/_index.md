## 📐 開幕テンプレ

### DT砲（初心者おすすめ）
```
配置:
□□□□■□□□□□
□□□□■■□□□□
□□□□□■□□□□
→ Tスピンダブル → テトリス の2連撃
```
**火力**: Tスピンダブル(4ライン)+テトリス(4ライン) = **合計8ライン送り**

### パフェ積み（上級者向け）
- 最初の10個のミノで完全消去を目指す
- 成功率: 約70%（パターン暗記で100%に近づく）
- **火力**: パーフェクトクリア = **10ライン送り**
