## 🗺️ マップ別攻略

### 🏙️ ダウンタウン（市街地マップ）

{{< stage-banner name="🏙️ ダウンタウン" theme="castle" >}}

| ポジション | 役割 | おすすめ武器 |
|-----------|------|------------|
| 🏢ビル屋上 | スナイパー | SR-338 |
| 🚗車の裏 | ライフル | AR-15 |
| 🏠室内 | エントリー | SMG-9 |
| 🛣️通り | サポート | LMG-M60 |

| ポジション | 役割 |
|-----------|------|
| ⛰️高台 | スナイパー（最重要ポジション） |
| 🪨岩陰 | ライフル |

## 💡 基本立ち回り

| | ルール |
|---|--------|
| 👁️ | ヘッドラインを常に意識（頭の高さにエイム） |
| 🚪 | 角を曲がる前にプリエイム（先にエイムを置く） |
| 🔊 | 足音を聞く。ダッシュは敵に位置バレ |
| 🗺️ | ミニマップを3秒に1回チェック |
| 👥 | 味方と一緒に動く（2人以上でクロスファイア） |
| 🔄 | 同じ場所に留まらない（キル後は移動） |
