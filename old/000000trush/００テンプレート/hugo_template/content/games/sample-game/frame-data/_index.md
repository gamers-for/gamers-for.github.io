## 📊 フレームデータの見方

| 用語 | 意味 |
|------|------|
| **発生** | 技が出るまでのフレーム数（小さいほど速い） |
| **持続** | 攻撃判定が残るフレーム数 |
| **硬直** | 技後の隙（マイナスほど不利） |
| **ガード時** | ガードされた時の有利不利 |
