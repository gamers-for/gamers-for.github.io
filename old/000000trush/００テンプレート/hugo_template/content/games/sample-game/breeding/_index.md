## 🥚 タマゴの仕組み

| 項目 | 内容 |
|------|------|
| 条件 | 同じタマゴグループの♂♀を預ける |
| 孵化歩数 | モンスターごとに異なる（1,000〜10,000歩） |
| 遺伝 | 個体値3つ+技+特性が遺伝可能 |

### 🧬 遺伝の仕組み

| 遺伝項目 | 条件 | 確率 |
|---------|------|------|
| 個体値3つ | 通常 | ランダム3つ |
| 個体値5つ | 💎運命の赤い糸持ち | 親から5つ引継 |
| 性格 | 📿かわらずの石持ち | 100%遺伝 |
| 技 | ♂親が覚えている技 | タマゴ技として遺伝 |
| 特性 | ♀親の特性 | 80%で同じ特性 |
| 隠れ特性 | ♀親が隠れ特性 | 60%で遺伝 |

### 🏆 最速厳選手順

1. 💎赤い糸 + 📿かわらずの石を用意
2. 4V以上の親を用意（レイドやフレンド交換）
3. 📿性格固定の親 + 💎高個体値の親を預ける
4. 🥚孵化→個体値チェック→良い子を親にループ
5. 🎯5V or 6V誕生で完了

**⏱️目安**: 約30分〜2時間で理想個体
