## 💻 基本コマンド

| コマンド | 効果 | 例 |
|---------|------|-----|
| `/gamemode` | ゲームモード変更 | `/gamemode creative` |
| `/tp` | テレポート | `/tp Player1 Player2` |
| `/give` | アイテム付与 | `/give @s diamond 64` |
| `/time` | 時間変更 | `/time set day` |
| `/weather` | 天気変更 | `/weather clear` |
| `/difficulty` | 難易度変更 | `/difficulty hard` |
| `/kill` | エンティティ削除 | `/kill @e[type=zombie]` |
| `/locate` | 構造物検索 | `/locate structure village` |

## ⚠️ 注意

| | 注意 |
|---|------|
| 🏆 | コマンド使用で**実績が無効化**される場合あり |
| 🌐 | マルチプレイではOP権限が必要 |
| 💾 | 大規模コマンドはバックアップ後に実行 |
