## 🏙️ 都市設計の基本

### 区画配置の原則

| 原則 | 内容 |
|------|------|
| 🏠住宅 | 商業と離しすぎない。騒音源から離す |
| 🏢商業 | 交通の要所に配置。住宅と近接 |
| 🏭工業 | 住宅から離す。高速道路沿い |
| 🌳公園 | 住宅地の中心に。地価UP効果 |

### 交通設計

| 道路タイプ | 交通量 | コスト | 用途 |
|-----------|--------|--------|------|
| 🛣️高速道路 | 超高 | 💰💰💰 | 都市間・工業接続 |
| 🚗大通り | 高 | 💰💰 | メイン動脈 |
| 🚗中道路 | 中 | 💰 | 住宅街のメイン |
| 🛤️路地 | 低 | 💰 | 住宅街内部 |
| 🚂鉄道 | 超高 | 💰💰💰 | 大量輸送 |
| 🚌バス | 中 | 💰 | 補助的移動 |
