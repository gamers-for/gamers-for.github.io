## 🔄 進化方法一覧

| 方法 | 条件 | 例 |
|------|------|-----|
| 📈レベルアップ | 特定Lvに到達 | Lv.16/Lv.36等 |
| 💎進化アイテム | 特定アイテムを使用 | 炎の石、雷の石 |
| 🤝通信交換 | 他プレイヤーと交換 | 交換で即進化 |
| 💕なつき度 | なつき度220以上+レベルアップ | 特定のモンスター |
| 📍特定場所 | 特定の場所でレベルアップ | 磁力の洞窟等 |
| ⏰時間帯 | 昼/夜にレベルアップ | 朝→太陽系、夜→月系 |
| 🔀分岐進化 | 条件で進化先が変わる | アイテムや性格で分岐 |
