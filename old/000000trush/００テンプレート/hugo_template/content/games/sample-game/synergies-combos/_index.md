## 💡 シナジー構築のコツ

| | コツ |
|---|------|
| 🎯 | 序盤で「軸」を決める（火力型/防御型/金策型） |
| 🔄 | 軸に合わないアイテムはスキップ |
| 🛒 | ショップでは軸アイテムを優先購入 |
| 📊 | 各階のボスに合わせて微調整 |
