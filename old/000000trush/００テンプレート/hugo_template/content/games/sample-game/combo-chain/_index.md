## 🧩 コンボの基本

| コンボ数 | 倍率 | 評価 |
|---------|------|------|
| 1コンボ | ×1.0 | - |
| 3コンボ | ×1.5 | 🔰初心者 |
| 5コンボ | ×2.5 | 📗中級 |
| 7コンボ | ×4.0 | 📘上級 |
| 10コンボ+ | ×7.0+ | 🏆達人 |

## 📐 パズル操作テクニック

### 基本テクニック

| テクニック | 難易度 | 説明 |
|-----------|--------|------|
| 横一列消し | ⭐ | 同じ色を3個横に揃える |
| L字消し | ⭐⭐ | 5個でL字に消して攻撃UP |
| 十字消し | ⭐⭐⭐ | 5個で十字に消して回復 |
| 斜め移動 | ⭐⭐⭐ | ドロップを斜めに持っていく |
| 端寄せ | ⭐⭐⭐⭐ | 不要色を端に追いやる |

### 盤面整理の考え方

```
悪い例:              良い例:
🔴🔵🔴🟡🔵🔴     🔴🔴🔴🔵🔵🔵
🟡🔴🔵🔴🟡🔵     🟡🟡🟡🟢🟢🟢
🔵🟡🔴🔵🔴🟡     🔴🔴🔴🔵🔵🔵
(色がバラバラ)       (色がまとまっている)
```

## 🏆 高コンボのコツ

| | コツ |
|---|------|
| 📏 | まず盤面を見て消せる場所を3箇所見つける |
| 🔄 | 端から整理していく（左上→右下） |
| ⏰ | 操作時間延長リーダーを使う |
| 🧠 | 動かす前に3手先まで考える |
