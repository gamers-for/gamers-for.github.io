## ⚔️ 拠点戦の仕組み

| 項目 | 内容 |
|------|------|
| 👥参加人数 | 最大50vs50 |
| ⏰時間 | 2時間 |
| 🏆勝利条件 | 相手の本拠地を制圧 or ポイント |
| 📅開催 | 毎週土曜 20:00〜22:00 |

## 🗺️ 拠点戦マップ

```
[攻撃側]
⚔️本陣 ─── 🏰中間拠点A ─── 🏰最終拠点
              │
           🏰中間拠点B ─── 🏰最終拠点
                              [防衛側]
```

## 👥 役割分担

| 役割 | 人数 | 任務 |
|------|------|------|
| ⚔️攻撃隊 | 20人 | 前線で拠点制圧 |
| 🛡️防衛隊 | 15人 | 自拠点の防衛 |
| 🏹遊撃隊 | 10人 | 裏取り・奇襲 |
| 💖支援隊 | 5人 | 回復・バフ |

## 💡 集団戦のコツ

| | コツ |
|---|------|
| 📢 | 指揮官の指示に従う（個人プレイNG） |
| 👥 | 固まって動く（散開すると各個撃破される） |
| 🎯 | ヒーラーを最優先で狙う |
| 🛡️ | タンクの後ろに隠れて前進 |
| ⏰ | リスポーン位置を意識（遠いと復帰が遅い） |
