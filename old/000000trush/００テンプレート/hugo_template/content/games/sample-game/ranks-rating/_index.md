## 🏅 ランクシステム

### 囲碁

| ランク | レーティング | 実力目安 |
|--------|------------|---------|
| 🔰30級〜20級 | 0-500 | ルールを覚えた |
| 📗15級〜10級 | 500-1000 | 基本定石を知ってる |
| 📘5級〜1級 | 1000-1500 | 中盤の判断ができる |
| 🏅初段〜三段 | 1500-2000 | 序盤〜終盤まで一通り |
| 🏆四段〜七段 | 2000-2500 | アマ上位。大会入賞レベル |
| 👑八段〜九段 | 2500+ | アマトップ / プロレベル |

### 将棋

| ランク | 実力目安 | おすすめ練習 |
|--------|---------|------------|
| 🔰10級以下 | 駒の動かし方OK | 1手詰め100問 |
| 📗5級〜1級 | 手筋を知ってる | 3手詰め+定跡 |
| 🏅初段〜三段 | 大局観がある | 5手詰め+棋譜並べ |
| 🏆四段以上 | 研究している | 7手詰め+AI研究 |
