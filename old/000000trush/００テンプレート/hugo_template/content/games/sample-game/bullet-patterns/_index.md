## 🎯 弾幕の種類

| パターン | 特徴 | 避け方 |
|---------|------|--------|
| 🔵放射状 | 中心から均等に広がる | 弾と弾の隙間を通る |
| 🌀渦巻き | 回転しながら広がる | 回転方向と逆に動く |
| ➡️指向性 | 自機狙い | 少し動いて誘導→戻る |
| 🔴ランダム | 不規則に散らばる | 画面下部で小さく動く |
| ⬛壁弾 | 隙間のある壁が迫る | 隙間を見つけて通過 |

## 👹 ボス攻略（ステージ4ボス）

### フェーズ1（体力100%〜70%）

| 攻撃 | パターン | 安全地帯 |
|------|---------|---------|
| 🔵通常弾 | 放射×3回 | ボスの真下やや左 |
| ➡️追尾弾 | 自機狙い×5 | 横に大きく動く |

### フェーズ2（体力70%〜30%）

| 攻撃 | パターン | 安全地帯 |
|------|---------|---------|
| 🌀渦巻き+🔵放射 | 複合 | 渦の回転に合わせて移動 |
| 💥レーザー | 3本固定 | レーザー間 |

### フェーズ3（体力30%〜0%）発狂

| 攻撃 | 対処 |
|------|------|
| 全パターン複合 | **💣ボム推奨**。無理せず使う |

## 💣 ボム使用タイミング

| | タイミング |
|---|----------|
| ✅ | 弾幕が避けきれない時（無敵時間で回避） |
| ✅ | ボス発狂フェーズ |
| ✅ | 残機が少ない時（保険） |
| ❌ | パターンで避けられる弾幕（もったいない） |
