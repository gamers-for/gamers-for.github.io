## 🔄 スキル回しとは

- 決まった順番でスキルを使い、DPS/HPS（回復量）を最大化する技術
- **GCD**（グローバルクールダウン）の間に**oGCD**（アビリティ）を挟む

## ⚔️ 近接DPS「剣聖」のスキル回し

### 基本コンボ（GCD）

```
① 斬撃 → ② 二段斬り → ③ 三段突き（コンボ完成→バフ付与）
↓ 繰り返し
① 斬撃 → ② 二段斬り → ③ 三段突き ...
```

### バースト回し（開幕）

```
ステップ:
1. 🔥バフスキル（oGCD）
2. ① 斬撃
3. ⚡攻撃アビリティ1（oGCD）
4. ② 二段斬り
5. ⚡攻撃アビリティ2（oGCD）
6. ③ 三段突き
7. 🌟奥義（oGCD）← 最大ダメージ
```

### DPS目安

| スキルLv | 目安DPS | 評価 |
|---------|---------|------|
| 初心者 | 8,000 | 🟢コンボを回せている |
| 中級者 | 12,000 | 🟡oGCDを挟めている |
| 上級者 | 15,000 | 🔴バーストを合わせている |
| 最適化 | 16,500+ | 🟣ロスなし完璧 |

## 💖 ヒーラー「白魔道士」のスキル回し

| 状況 | 優先スキル |
|------|----------|
| 通常時 | 攻撃魔法でDPS貢献 |
| タンクHP減少 | 単体回復（GCD） |
| 全体ダメージ後 | 全体回復（oGCD優先→GCDで補助） |
| 緊急時 | 即時回復アビリティ |

**鉄則**: oGCD回復を使い切ってからGCD回復を使う（DPSロスを最小化）
