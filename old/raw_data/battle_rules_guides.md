## 1. ナワバリバトル

**ソースURL:** https://game8.jp/splatoon3/485632

### ルール概要

- **勝利条件:** 塗ることができる床をできるだけ多く塗ったチームの勝ち
- **制限時間:** 3分（残り30秒が最も重要）
- **対象:** すべての塗れる床が評価対象（壁は対象外）。自陣近くも含む
- **備考:** 最も基本的なルール。ランク1から参加可能

### 攻略ポイント & 勝つコツ

#### 初動の動き方

- エリア真ん中と自陣近くに分かれて塗る
- 味方のスポーン地点を確認し、塗る場所の重複を避ける
- 味方がどこを塗ろうとしているかを想像し、塗り残りそうな場所をカバー

#### 中盤の立ち回り

- 自陣近くをある程度塗ったら、真ん中で相手の進行を抑えている味方のカバーに向かう
- マップを見ながら相手の人数の多くが前線に来ている場合は早めに行く
- 有利時は相手の進行を抑え、相手の抜けに注意する
- 不利時は自陣近くの残った塗りをしながらスペシャルを溜める

#### 終盤のコツ（残り30秒が最重要）

- **残り30秒が一番大事** -- この局面では絶対にデスを避ける
- 相手の隙をついてキルを取る
- ラスト1秒くらいでボム系のサブウェポンを投げる（タイムアップ前に投げたボムはタイムアップ後でも爆発して塗りポイントを取る）

#### 塗りのコツ

- 「塗るところは綺麗に塗る」-- 小さな塗り残しが積み重なると勝敗に影響
- 初動で過度に塗り完成させない（スペシャル溜めのスペースを確保）
- 普段あまり塗らない自陣近くも綺麗に塗る

### おすすめ武器

「塗りと少しのキルができる武器が一番強い」

| 評価 | 武器名 |
|------|--------|
| X | バケットスロッシャー、N-ZAP85、シャープマーカー、LACT-450 |
| S+ | ジムワイパー、ダイナモローラー、スプラマニューバー 他多数 |

### 立ち回り方

| 段階 | 役割 |
|------|------|
| 自陣周辺 | 初動で自陣近くを綺麗に塗り、味方不足箇所をカバー |
| 前線 | 真ん中で相手の進行を抑えている味方をサポート |
| 有利時 | 前線をキープし、相手の抜けを警戒 |
| 不利時 | スペシャルを貯めながら耐える |

## 2. バンカラマッチ概要 & ウデマエシステム

**ソースURL:** https://game8.jp/splatoon3/477905

### バンカラマッチとは

勝利してウデマエと呼ばれるランクを上げていくことを目的としたモード。4つのガチルール（ガチエリア、ガチヤグラ、ガチホコ、ガチアサリ）で戦う。

### 参加条件

- ランク10から参加可能
- スプラ2のセーブデータがあれば最初から参加可能

### ウデマエシステム

| ランク帯 | 詳細 |
|----------|------|
| C- | 最低ランク |
| C | |
| C+ | |
| B- | |
| B | |
| B+ | |
| A- | |
| A | |
| A+ | |
| S | |
| S+ | S+0 ~ S+50（S+0到達後、数字が増えていく） |

- **リセット機能:** シーズン中1回のみ、任意でウデマエをリセット可能（2022年11月30日アップデートより）

### チャレンジ vs オープン

| 項目 | チャレンジ | オープン |
|------|-----------|---------|
| プレイ形式 | 完全ソロプレイ | フレンドと一緒にプレイ可能 |
| ポイント増減 | 勝った時のポイント増加量が大きい | ポイント増減量が少ない |
| 終了条件 | 5勝または3敗で終了 | まったりプレイ可能 |
| 特徴 | ガチ向け | カジュアル向け |

### ポイント計算式

**獲得ポイント = 勝利数ポイント + 表彰ポイント - 参加費**

- 参加費はウデマエごとに異なる

### バンカラマッチ全体の攻略コツ

1. **スペシャルを活用** -- 打開のためにあると言っても過言ではないほど重要。不利な状況での反撃に活用
2. **味方との連携** -- 人数有利を押し付けるように打開する
3. **武器知識** -- 自分と相手の武器の射程と威力を理解する
4. **厄介な相手を優先** -- チームの障害となる相手を倒す
5. **得意なルール・ステージで挑戦** -- 自分の得意分野を活かす
6. **ルール関与** -- ホコを進めたり、カンモンを突破するなど、ルールに則った行動
7. **不利時の慎重さ** -- 塗りが不利な状況での無理な攻撃を避ける
8. **相手スペシャル潰し** -- 有利な状況で相手のスペシャル使用を阻止

## 3. ガチエリア

**ソースURL:** https://game8.jp/splatoon3/397597

### ルール概要

- **勝利条件:** 指定されたエリアを自チームのインクで塗りつぶし、カウントをより多く進めたチームの勝利
- **制限時間:** 5分
- **延長条件:** カウントが負けているチームがガチエリアを確保している場合に延長。カウントが逆転されるか中立状態で一定時間経過すると延長終了

### ペナルティシステム

- こちらがエリアを確保してから、敵に塗り変えられるとペナルティカウントが付く
- 進めたカウントが大きいほどペナルティも増加
- 一度の確保時間を長く保つことが重要

### 攻略ポイント & 勝つコツ

#### 初動の動き方

- 基本的にはエリアに向かう
- マップを確認して敵の配置を認識
- 自分だけエリアに向かわず自陣を塗っていると、中央で人数不利が起こり味方がキルされるリスクが高まる

#### 有利時の立ち回り

- カウントリードを意識
- チームにとって厄介な敵を倒す
- 相手にスペシャルを使わせない

#### 不利時の立ち回り（打開のコツ）

- 生存しつつ打開の準備をする
- スペシャルを溜める
- 味方の復帰を待つ
- 味方と打開のタイミングを合わせる

#### 抑えのコツ

- エリア内で倒されない（倒されると相手のインクが飛び散り、相手のエリア塗りを手助けしてしまう）
- 相手にスペシャルを使わせないよう圧をかける

#### NG行動

- 不利な状況で無理に前に出る
- エリア内で倒される
- 対面中の味方にスーパージャンプする（沼ジャン）

### おすすめ武器

基本的に「塗りが強い武器が強力」

| 評価 | 武器名 |
|------|--------|
| X | ジムワイパー、スプラマニューバー、バケットスロッシャー、シャープマーカー、スプラシューター、ボトルガイザー |
| S+ | N-ZAP85、プライムシューター、52ガロン、クーゲルシュライバー、ノーチラス47 他 |

### 立ち回り方（前衛/中衛/後衛）

| 役割 | 動き方 |
|------|--------|
| 前衛（短射程） | エリア周辺で敵を排除、エリア確保を維持 |
| 中衛 | エリアの塗り維持と前衛のカバー |
| 後衛（長射程） | 安全な位置からエリアを塗りつつ敵を牽制 |

## 4. ガチヤグラ

**ソースURL:** https://game8.jp/splatoon3/478554

### ルール概要

- **勝利条件:** ヤグラに乗り、相手のゴールまで進める。相手ゴール到達で即勝利、または5分終了時点でカウントが多いチームの勝利
- **制限時間:** 5分
- **基本:** 誰か1人でもヤグラに乗っている間、ヤグラが進む。誰も乗っていないヤグラは中央に戻る

### カンモンの仕組み

- 進行ルート上に複数のカンモンポイントが存在
- ヤグラが到達時に誰かが乗り続けないと突破できない
- 一度突破したカンモンは2回目以降素通りできる（押し返された場合も進めやすくなる）

### 攻略ポイント & 勝つコツ

#### 初動の動き方

- 初動はヤグラに向かうが、キルを優先して人数有利を作ることが重要
- 人数有利なしでヤグラに乗っても集中砲火で倒されるだけ
- まず人数有利を作ってからヤグラに乗る

#### 有利時の立ち回り

- 前線を上げて相手をヤグラに近づけない距離感を保つ
- 短射程は周囲で守り、長射程がヤグラ上に乗る戦略が有効
- 前線を上げすぎてヤグラから離れるのはNG

#### 不利時の立ち回り

- 相手のヤグラ上を守っているプレイヤーを倒す
- 人数有利を再度作ることでヤグラ奪還のきっかけを作る
- 焦って無理に突っ込まない

#### ヤグラ乗車のコツ

- 乗る前にヤグラの裏に敵が潜伏していないか確認
- ヤグラ上ではヒト状態で柱を盾にする
- 膠着状態では一瞬だけ乗ってヤグラを動かすテクニックが有効

### おすすめ武器

| 評価 | 武器名 |
|------|--------|
| S+ | ジムワイパー、スプラマニューバー、ロングブラスター、ホットブラスター、プライムシューター |

### 立ち回り方（前衛/中衛/後衛）

| 役割 | 動き方 |
|------|--------|
| 前衛（短射程） | ヤグラ周囲で敵を排除、護衛 |
| 中衛 | ヤグラ進行ルートの塗り確保とカバー |
| 後衛（長射程） | ヤグラ上に乗りつつ遠距離から敵を牽制 |

## 5. ガチホコバトル

**ソースURL:** https://game8.jp/splatoon3/478551

### ルール概要

- **勝利条件:** ガチホコを相手のゴールまで持って行けたら勝利。時間切れの場合はカウントが多い（ゴールに近い）チームが勝利
- **制限時間:** 5分
- **ホコ爆発:** ホコを60カウント持ち続けると爆発
- **延長条件:** 時間切れ時に負けているチームがホコを所持している場合、延長戦

### カンモンの仕組み

- スプラ3から追加された機構
- 中央からゴールまでの間に2箇所のカンモンが存在
- どちらかに一度ホコを置いてからでないと、その先のカウントが進まない

### 攻略ポイント & 勝つコツ

#### 基本方針

- 確実に進められるときだけ進む
- 全員で前線を上げる
- マップ構造とホコルートを把握
- 自陣ゴール手前が敵に塗られていたら塗り返す

#### 初動の動き方

- 基本的には中央のガチホコに向かう
- 自陣を塗っている時間は避ける
- ただしスペシャル増加量アップを積んで初動スペシャルを狙う場合は例外

#### ホコ持ちの立ち回り

- 守りやすい場所で倒れる意識が重要
- 短射程が前衛、長射程がホコ持ちが基本（ガチホコショットは後衛に近い役割を持つため）
- 有利時はホコ持ちが進みやすいよう全員で前線を上げる

#### 不利時の立ち回り

- むやみに突っ込まない
- 広く展開して複数の射線を作って迎撃

#### NG行動

- 一人で突っ込む
- 自陣にホコを持ち帰る
- 自陣でホコを持って時間を無駄にする
- ホコを割ったまま放置
- 対面中の味方へのスーパージャンプ（沼ジャン）

### おすすめ武器

| 評価 | 武器名 |
|------|--------|
| X | ジムワイパー、スプラマニューバー、バケットスロッシャー、シャープマーカー、ボトルガイザー |
| S+ | ドライブワイパー、N-ZAP85、プライムシューター、52ガロン、スプラシューター 他 |

### 立ち回り方（前衛/中衛/後衛）

| 役割 | 動き方 |
|------|--------|
| 前衛（短射程） | 敵を排除してホコ持ちの進路を確保 |
| 中衛 | ホコ持ちが進みやすいようフィールドを塗りつつ前線を上げる |
| 後衛（長射程） | ホコを持ち、ガチホコショットで後衛的に戦う |

## 6. ガチアサリ

**ソースURL:** https://game8.jp/splatoon3/480377

### ルール概要

- **勝利条件:** 相手ゴールのアサリバリアを壊し、アサリを投げ入れてカウントを0にする。または5分終了時点でカウントをより進めたチームが勝利
- **制限時間:** 5分
- **アサリの仕組み:** 8個のアサリを集めるとガチアサリに変化。ガチアサリの位置は味方・相手両方に見える

### バリアの壊し方

- ガチアサリを相手のバリアにぶつけるとバリアが壊れる
- バリアが壊れるとチャンスタイム開始

### カウンターアサリの使い方

- 相手のチャンスタイム終了後、ゴールにカウンターアサリが出現
- 自チームが不利な時やゴールに近い時に使うと効果的
- むやみに拾わない

### 攻略ポイント & 勝つコツ

#### 必須級のコツ

- アサリ・ガチアサリの飛距離を知っておく（どこからゴール可能か判断）
- 無理にゴールせず、前線を徐々に押し上げる意識
- 1回のゴールでなるべく多くのカウントを進めるため、複数人がアサリを持って攻める

#### アサリの集め方

- 7個まで集めてガチアサリにしない（位置がバレるため）
- ガチアサリに進化させるのは前線を押し上げた後
- アサリを大量に持ちながら何度もデスしないよう注意

#### 初動の動き方

- 味方と協力してキルをする → 人数有利を作る → 前線を押し上げる

#### ゴール時

- 盤面制圧力の高いスペシャルを使ってゴール
- 前線を上げていないのに無理やりゴールに向かうのはNG

#### 防衛時

- 相手のガチアサリを持っているプレイヤーを集中的に止める

### おすすめ武器

- ゴールにアサリを入れやすい武器が強力
- サブにシールドを持つ武器やスペシャルにグレートバリアを持つ武器が有効

### 立ち回り方（前衛/中衛/後衛）

| 役割 | 動き方 |
|------|--------|
| 前衛（短射程） | 敵を排除し前線を押し上げ、ゴール前を確保 |
| 中衛 | アサリを集めつつ前衛のカバーとゴールサポート |
| 後衛（長射程） | 敵のガチアサリ持ちを狙い撃ち、防衛時は相手を寄せ付けない |

## 7. Xマッチ

**ソースURL:** https://game8.jp/splatoon3/482084

### Xマッチとは

- S+より上の戦い、レートを競うモード
- 実質的にS+の上のランク帯
- 完全ソロで挑むモード（グループ参加不可）

### 参加条件

- バンカラマッチのウデマエがS+0以上で参加可能
- 最速で挑戦したい場合はS+10以上を目指すことが推奨

### 対戦ルール

- バンカラマッチの4つのルール（ガチエリア、ガチヤグラ、ガチホコバトル、ガチアサリ）で戦う
- 各ルールで別々のXパワーを競う

### Xパワーの仕組み

- 最初の5試合でXパワーの計測を行う（計測戦）
- 計測戦後は3勝するか3敗するまでバトルを行い、結果によってXパワーが変動
- シーズンごとに計測し競う

### 地域マッチング

- グループ・ヤコ: 日本・香港・韓国など
- グループ・アロメ: アメリカ大陸・ヨーロッパ

### 報酬

- レートが上位帯になると名前の上に王冠がつく報酬

### 実装日

- 2022年12月1日午前9時に実装

## 8. サーモンラン

**ソースURL:** https://game8.jp/splatoon3/480290

### ルール概要

- 4人でチームを組み、シャケを倒して金イクラを集めるバイトモード
- オオモノシャケを倒すと金イクラが入手できる
- 各ウェーブでノルマを達成する必要がある
- 3ウェーブ全てをクリアするとバイト成功

### 攻略ポイント & コツ

#### 1. 厄介なオオモノシャケを優先撃破

- 全滅リスクが高い敵から倒す
- テッキュウは最優先
- カタパッド、タワーも危険度が高い

#### 2. 武器による役割分担

| 射程 | 役割 |
|------|------|
| 短射程 | 前線でザコシャケ処理とオオモノシャケ撃破 |
| 長射程 | 後方からオオモノシャケ狙撃と前線フォロー |

#### 3. 寄せ（ルアリング）テクニック

- 一部のオオモノシャケは近づかないと寄ってこない
- 効率的な金イクラ納品のため、コンテナ付近までオオモノシャケを引き寄せてから倒す

#### 4. 味方救助の優先

- 倒された味方にすぐインクをかけて復活させる
- 一人がダウンすると全滅へのドミノ倒しが始まる

#### 5. スペシャルウェポンの使用

- 出し惜しみせず、危険時にすぐ発動する
- 3ウェーブで2回の使用機会（エクストラウェーブで1回追加）

### 特殊イベント（WAVE）

| イベント名 | 概要 |
|-----------|------|
| 水位変化 | 水位が上下し、戦えるエリアが変化 |
| ヒカリバエ（ラッシュ） | 大量のシャケが押し寄せる。高台集合が基本 |
| 霧 | 視界が悪くなり、オオモノシャケが大量出現 |
| グリル発進 | グリルが特定プレイヤーを追尾。おとり役と攻撃役に分かれる |
| ドスコイ大量発生 | 巨大なドスコイが大量に出現 |
| キンシャケ探し | 間欠泉からキンシャケを探す。正解の間欠泉を見つける |
| ハコビヤ襲来 | 空からハコビヤが金イクラを運んでくる。落として納品 |
| ドロシャケ噴出 | ドロシャケが噴出する。噴出口を塗って止める |
| 巨大竜巻 | 巨大な竜巻が発生。竜巻から離れつつシャケを処理 |

### 報酬

- 3ウェーブクリアで金イクラ納品数に応じた報酬獲得
- オカシラシャケ撃破時は「ウロコ」がもらえる
- ウロコでオキモノなど特殊報酬と交換可能
- ランク上昇、ギア、カプセルなども獲得

| X（最優先） | テッキュウ、カタパッド、タワー |
| S+ | コウモリ |
| S | ヘビ、ハシラ |
| A | バクダン、モグラ |
| B | ナベブタ、テッパン、ダイバー |

**優先順位の基準:**
- 遠距離・広範囲攻撃でこちらの行動が制限されるものは高い
- 被害が全体に及ぶものは高い
- 放置時の塗り被害範囲が広いものも優先度高め

### 各オオモノシャケの詳細な倒し方

#### テッキュウ（優先度: X）

- **倒し方:** メインでダメージを与えて倒す
- **特徴:** 海岸付近に発射台を設置し砲弾を発射する
- **コツ:** こちらから倒しに行く必要がある。放置すると継続的にダメージを受ける

#### カタパッド（優先度: X）

- **倒し方:** ミサイル発射口にボムを投げ込む（両肩を破壊で撃破）
- **別手段:** ミサイル発射口にエクスプロッシャーの弾を当てる
- **コツ:** 両方のミサイル発射口にボムを入れる必要がある。片方ずつ破壊可能

#### タワー（優先度: X）

- **倒し方:** 体を攻撃して最上段まで段を落とすと倒せる
- **特徴:** 追尾するレーザー攻撃を行う
- **コツ:** 危険度が非常に高いため、見つけ次第すぐに処理する

#### コウモリ（優先度: S+）

- **倒し方:** 弾を撃った後に現れる本体を攻撃する
- **別手段:** 撃ち返した弾を当てると一撃で倒せる
- **コツ:** 弾を撃つタイミングで本体が露出するのを狙う

#### ヘビ（優先度: S）

- **倒し方:** 尻尾の操縦手をメインで攻撃する
- **特徴:** 非常に長い体が移動を制限する
- **コツ:** 体ではなく尻尾の操縦手を狙うこと

#### ハシラ（優先度: S）

- **倒し方:** メインで倒せる
- **コツ:** 柱上に登ると攻撃しやすい。柱のインクレールを塗って登る

#### バクダン（優先度: A）

- **倒し方:** 頭にボムを作り始めたら、ボムを飛ばす前にインクを当てて爆発させる
- **注意:** 体に攻撃は通らない
- **コツ:** ボムを作るタイミングを見計らって頭部を集中攻撃

#### モグラ（優先度: A）

- **倒し方:** 攻撃地点にボムを置いて飲み込ませる
- **特徴:** ウキで居場所を確認できる
- **コツ:** モグラが攻撃しようとする場所にボムを先置きする

#### ナベブタ（優先度: B）

- **倒し方:** 本体にインクを当てれば倒せる
- **コツ:** 落下を誘って円盤の上に乗ると楽に倒せる

#### テッパン（優先度: B）

- **倒し方:** 盾にインクを当て続けると動きが止まる。動かなくなったら後ろから本体を攻撃
- **コツ:** 正面から盾を攻撃して止めてから背後に回り込む

#### ダイバー（優先度: B）

- **倒し方:** メインで本体へダメージを与えられる
- **コツ:** リング内のエリアを塗り替えると飛び込めなくなる
