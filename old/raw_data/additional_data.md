# 初心者攻略（Game8）

## 序盤の進め方
1. キャラクター作成（後から変更可能）
2. チュートリアル完了
3. 設定調整 - **ジャイロ操作をON**に推奨
4. 試し打ち練習 - 敵の倒し方を習得
5. ヒーローモード - ストーリーモードで基礎を学ぶ
6. オンライン対戦参加
7. 武器選択 - 複数回対戦後に決定

## おすすめ初心者武器
- **スプラシューター** - 解放ランク2、扱いやすくキルも取りやすい
- **バレルスピナー** - 長射程で後方から安全にキル
- **N-ZAP85** - 塗り性能と支援能力が高い

## 基本操作
- Lスティック: 移動
- Rスティック: 視点移動
- Bボタン: ジャンプ
- ZRボタン: メイン武器射撃
- ZLボタン: イカセンプク状態
- Xボタン → Aボタン: スーパージャンプ
- R(右)スティック押し込み: スペシャル発動

## 重要テクニック
- **イカロール** - 進行方向と逆にスティック＋Bでアーマー付与
- **イカノボリ** - 壁でBボタン長押しで上昇
