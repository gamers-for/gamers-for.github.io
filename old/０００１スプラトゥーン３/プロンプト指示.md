# フェーズ3 コンテンツ生成プロンプト指示

## 必ず守ること

### 禁止事項
1. **他サイト名を一切書かない** — Game8, GameWith, Altema, ゲーム8, ゲームウィズ, アルテマ、「3サイト」「攻略サイトの評価を統合」等の文言は一切禁止。HTMLコメント内でも禁止。
2. **テーブル内に画像を入れない** — `| ![img](url) テキスト |` のようにテーブルセル内にアイコンとテキストを同居させない。テーブルはテキストのみ。画像はテーブル外で十分なスペースを取って配置する。
3. **テンプレ文章で埋めない** — 「広め」「狭め」のような抽象的・汎用的な記述は禁止。スクレイプした実データに基づく具体的な情報を使う。

### 必須事項
1. **3サイトの情報の和集合（union）で作る** — どれか1つにしかない情報も全て含める。情報量を最大化する。
2. **スクレイプしたHTMLの実データを使う** — テンプレではなく、HTMLから抽出した具体的なステージ名・イベント名・アプデ内容・フェス結果等を含める。
3. **画像・アイコンを活用する** — テーブル外の適切な位置に、十分なスペースを取って配置する。
4. **レイアウトに余裕を持たせる** — 改行を入れて要素間に空白を確保。狭いスペースに詰め込まない。

### 確認チェックリスト（生成後に必ず実行）
```bash
# 1. 他サイト名チェック（0件であること）
grep -ri "game8\|gamewith\|altema\|3サイト\|3大攻略" content/

# 2. テーブル内画像チェック（0件であること）
grep -r "| !\[" content/

# 3. フェーズ3プレースホルダチェック（0件であること）
grep -r "フェーズ3で記入" content/
```

### コンテンツ品質基準
- ステージページ: 各ステージの**具体的な特徴**（中央の高台、左右の通路、壁の位置等）を記述
- イベント/フェス: **過去の開催実績**（テーマ、結果）を含める
- アップデート: **具体的な変更内容**（武器バランス調整、新武器追加等）を含める
- 武器評価: **数値データ**（ダメージ、射程、キルタイム）と**具体的な立ち回り**を記述
- ギア: **具体的な効果量**と**武器種ごとの推奨組み合わせ**を記述
