# Gamers-For コンテンツ制作方針

## 基本原則：合法性の根拠

| 要素 | 手法 | 法的根拠 | 判定 |
|---|---|---|---|
| HTML構造 | コピー | HTML構造に著作権なし | OK |
| 文章 | 要約・書き直し | 表現を変えれば著作権クリア | OK |
| 画像 | プレースホルダー → 後日差し替え | 何も侵害していない | OK |
| 数値データ | そのまま使用 | 事実情報に著作権なし | OK |
| アプデ履歴 | そのまま使用 | 事実の羅列に著作権なし | OK |

---

## 公開フェーズ

### Phase 1: 即日公開（テキスト先行）

- HTML構造コピー + 文章書き直し + 画像プレースホルダー
- 500タイトル一気に公開
- Googleインデックス開始
- プレースホルダーでもテキスト情報が充実していればGoogleは評価する

### Phase 2: 画像を段階的に埋める

| 優先度 | 対象 | 対応 |
|---|---|---|
| S（25タイトル） | 主力タイトル | Fandom + ゲーム素材で画像埋め |
| A | 準主力 | 順次対応 |
| B | その他 | 後回し or プレースホルダーのまま |

画像がなくても検索順位は上がる。画像は後から入れれば表示が豪華になり、ユーザー滞在時間が伸びてさらにSEO向上。

---

## 画像取得方針

### アイコン系（キャラ・武器・アイテム）

- **ソース**: ゲーム公式素材 / ファンキット / データマイニング済み素材
- **加工**: WebP変換 + リサイズ + GF透かし
- **理由**: 他サイトのアイコンも元はゲーム素材。同じソースから取れば問題なし

### スクショ・ステージ画像・キャプチャ系

- **ソース**: Fandom Wiki
- **加工**: WebP変換 + GF透かし
- **理由**: ファンが撮ったスクショがCC-BY-SA等で公開されている

---

## Fandom画像のライセンス対応

FandomのライセンスはWikiごとに異なる。大半は **CC-BY-SA**。

### CC-BY-SA の条件

1. **出典を明記** → ページ下部に小さく「Images: Fandom Wiki」でOK
2. **同じライセンスで公開** → 画像部分だけCC-BY-SAにすればOK

### 対応方法

- フッターに一行「一部画像はコミュニティWikiより引用」と記載
- 他サイトはこれすらやっていないが、Gamers-Forはちゃんとやる（安全策）
